## 核心发现摘要

### 优势
- 主打产品派格宾(PEG-rhG-CSF)在长效G-CSF市场建立领先地位
- 专利保护期至2030年,中期竞争压力有限
- 研发管线聚焦血液学和肿瘤学,市场空间大
- 2020-2023年营收CAGR达25%,增长动能强劲

### 劣势
- 产品线过度依赖派格宾单一产品(占比超80%)
- 新产品特泊替尼面临激烈竞争
- 销售费用率长期高位(40-50%),影响利润率
- 医保控费压力持续加大

### 投资建议
- 估值合理(P/E 35x),符合成长股估值区间
- 建议等待新产品数据或估值回调至30x以下
- 重点关注特泊替尼上市进度及派格宾市场份额

## 研究方法论

本报告遵循以下研究原则:

1. **多源验证**: 利润表与现金流量表交叉验证
2. **同业对比**: 与安科生物、长春高新等同行对比
3. **熊市压力测试**: 构建多种下行情景
4. **引文质量**: 所有事实陈述均标注来源(A-E评级)
