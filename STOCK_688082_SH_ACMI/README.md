## 投资逻辑速览

### 核心优势 ✓

1. **技术领先**: SAPS™、TEBO™等自主核心技术国际领先
2. **平台化布局**: 从清洗设备扩展至电镀、炉管、PECVD等多产品线
3. **国产替代**: 受益于中国半导体设备国产化大趋势
4. **高增长赛道**: 清洗设备市场2025-2030年CAGR 7.74%，中国CAGR 25%
5. **优秀管理层**: 创始人王晖技术背景深厚，持股稳定

### 核心风险 ⚠

1. **估值偏高**: PE 44.6x，PS 10.75x，估值处于高位
2. **现金流风险**: 2025年经营性现金流骤降
3. **客户集中**: 依赖主要半导体制造商客户
4. **地缘政治**: 中美科技摩擦影响
5. **技术迭代风险**: 半导体设备技术更新快

### 估值与建议

| 评估维度 | 评级 | 说明 |
|----------|------|------|
| **生意质量** | 🟢 优秀 | 高毛利、高ROE、强护城河 |
| **财务质量** | 🟡 良好 | 营收高增长，但现金流恶化 |

| **估值** | 🔴 偏高 | PE 44.6x，缺乏安全边际 |

**综合评级**: 🟢🟢🟡 **买入 - 适度乐观**
