## 核心投资亮点

- 2024年营收10.76亿元（-0.90%），归母净利润9547万元（+15.68%）
- 2024年扣非净利润9041万元（+21.50%）
- 行业内产业链最完整、产品种类最齐全
- 拥有国家专利110项，12项核心技术
- 通过十多项体系及产品认证

## 主要风险提示

- 🟡🟡 2025年一季度营收2.1亿元（-14.48%），归母净利润1292万元（-15.49%）
- 🟡🟡 2025年一季度扣非净利润1112万元（-23.48%）
- 🟡🟡 毛利率仅20%左右，盈利空间有限
- 🟡🟡 ROE 9.30%，股东回报率一般
- 🟡🟡 应收账款/利润达604.28%，应收账款风险极高
- 🟡🟡 货币资金/流动负债仅17.63%，现金流状况需关注
- 🟡🟡 近3年经营性现金流均值为负
