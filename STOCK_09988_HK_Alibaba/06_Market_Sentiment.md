# 阿里巴巴(09988.HK)市场情绪分析报告

## 执行摘要

当前市场对阿里巴巴(09988.HK)呈现**高度分歧**的态度，多头观点聚焦于AI技术领先、云业务加速增长、估值便宜及股东回报丰厚四大优势；空头观点则担忧电商份额流失、拼多多抖音竞争加剧、新业务巨额投入拖累盈利以及监管风险持续。

**核心争议点**：电商份额能否守住30%关键关口、云业务能否持续高增长、AI能否转化为实际收入。

**市场分歧评级**：⚠️ **高度分歧** (Bull-Bear分歧指数: 7.5/10)

---

## 一、多头观点(Bull Case)

### 1.1 AI技术领先优势显著

**核心论点**：阿里巴巴在AI领域的技术投入已开始产生实际回报，AI相关收入连续多个季度实现三位数增长。

**关键证据**：
- **AI收入爆发式增长**：2026财年Q1(2025年Q2)阿里云营收333.98亿元，同比增长26%，其中AI相关收入连续多个季度实现三位数增长，外部商业化收入中AI贡献已超过20%【网易,2025-09-01】[C级]
- **资本开支创历史新高**：2025年Q2资本开支386亿元，同比增长220%，环比增长57.1%，创单季历史新高。过去四个季度累计投入超1000亿元【网易,2025-09-01】[C级]
- **大模型技术领先**：通义千问2.5版本性能全面赶超GPT-4 Turbo，1100亿参数开源模型超越Llama-3-70B成为开源领域最强大模型【极客公园,2024-05-15】[C级]
- **三年3800亿投入计划**：公司宣布未来三年计划投入3800亿元用于云和AI基础设施建设【搜狐,2026-02-15】[C级]

**市场反应**：2025年9月1日阿里巴巴盘中一度涨近19%，创2022年3月以来最大单日涨幅，反映市场对AI叙事的认可【网易,2025-09-01】[C级]

### 1.2 云业务加速增长，市场份额提升

**核心论点**：阿里云在AI需求推动下重返双位数增长轨道，市场份额稳居第一。

**关键证据**：
- **增速持续提升**：2026财年Q2云智能集团收入398.24亿元，同比增长34%(Q1为26%)，剔除并表子公司影响的外部收入增速约29%【网易,2025-12-01】[C级]
- **市场份额扩大**：AI拉动阿里云市场份额扩大至36%，在中国AI云市场处于领先地位【搜狐,2026-02-11】[C级]
- **盈利能力改善**：2026财年云智能集团经调整EBITA利润增长31%，占集团合并经调整EBITA利润约13.7%【阿里巴巴财报,2025-05-14】[A级]
- **预期兑现**：CEO吴泳铭此前承诺"2025财年下半年商业化营收重返双位数增长"已提前实现【极客公园,2024-05-15】[B级]

**机构观点**：瑞银认为阿里云的投资强度和持续性指引"超出市场预期"【网易,2025-12-01】[B级]

### 1.3 估值便宜，安全边际充足

**核心论点**：当前估值处于历史低位，PE仅12倍左右，具备显著安全边际。

**关键证据**：
- **PE估值倍数**：目前对应估值仅有10-12倍PE，远低于历史平均和互联网同行【雪球,2025-01-04】[D级]
- **券商目标价**：中信证券给予234港元目标价(2026-02-23)、建银国际169.1港元、美银证券114港元，当前股价存在显著上升空间【证券之星,2024-2025】[B级]
- **机构评级一致**：近90天内共有18-27家投行给出买入评级，目标均价115-166港元【证券之星,2024-2025】[B级]
- **市值折价**：2019年回港上市时市值超4万亿港元，截至2025年11月市值2.89万亿港元，跌幅显著【网易,2025-12-01】[C级]

**分析师观点**：雪球分析师指出"阿里巴巴堪称是价值投资的典范，2026年经调整利润大概率能到2000亿港币，给25-30倍PE就是5万亿以上估值"【雪球,2025-01-04】[D级]

### 1.4 股东回报丰厚，回购分红力度大

**核心论点**：公司持续加大回购和分红力度，股东回报率超越腾讯等互联网同行。

**关键证据**：
- **回购规模**：2025财年已回购119亿美元股份，使流通股净减少5.1%。自2024年8月通过普通决议案以来累计回购2.62亿股，占总股本1.4%【阿里巴巴财报,2025-05-14】[A级]
- **分红力度**：董事会批准派发年度和特别股息合共46亿美元【阿里巴巴财报,2025-05-14】[A级]
- **股东回报率**：广发证券估算2024年股东回报率(分红+回购，剔除股权激励)逾11%；招商证券测算FY2024约为9%，FY2025预计6.2-8.1%或更高【广发证券/招商证券,2024】[B级]
- **剩余回购额度**：截至2024年6月，回购计划下仍余261亿美元回购额度，有效期至2027年3月【网易,2024-07-05】[B级]
- **现金储备充足**：现金储备达5700-5856亿元，为持续回购提供保障【搜狐/网易,2025】[C级]

**机构观点**：瑞银指出"阿里加快回购的步伐远远超出预期"，显示管理层对股东回报承诺的稳健执行【网易,2024-07-05】[B级]

## 二、空头观点(Bear Case)

### 2.1 电商市场份额持续流失，竞争格局恶化

**核心论点**：拼多多和抖音电商快速崛起，持续蚕食阿里巴巴市场份额，电商基本面临严峻挑战。

**关键证据**：
- **份额下滑**：阿里巴巴电商市场份额从2013-2014年的80%以上，一路下滑至2024年的33%，拼多多升至19%，抖音电商升至18%，京东17%【网易,2025-07-04】[C级]
- **GMV增速停滞**：阿里巴巴连续四年保持8万亿成交总额，但增速几乎停滞。2024年双十一交易额710亿元，增速仅10%【百家号,2025-02-24】[C级]
- **用户差距缩小**：拼多多年活跃买家9亿，与阿里淘天集团9.15亿用户差距从2017年的3.86亿缩小至1.26亿【网易,2025-07-04】[C级]
- **下沉市场失守**：2025年618数据，拼多多在下沉市场订单占比保持52%，阿里与京东通过淘特、京喜跟进但未能完全缩小差距【网易,2025-07-04】[C级]
- **流量劣势**：抖音电商用户日活超4亿，用户日均使用时长超120分钟，是阿里的2.3倍。直播电商复购率比传统电商高40%【网易,2025-07-04】[C级]

**行业观察**：杭州服装厂老板直言"现在的年轻人更喜欢在抖音上刷视频购物，或者去拼多多比价，谁还守着淘宝？"【百家号,2025-02-24】[C级]

### 2.2 拼多多、抖音竞争加剧，盈利压力加大

**核心论点**：为应对拼多多和抖音竞争，阿里被迫加大补贴和用户投入，导致短期利润大幅下滑。

**关键证据**：
- **闪购补贴大战**：淘宝闪购推出500亿元平台直补计划，12个月内通过店铺补贴、商品补贴、配送补贴等方式投入，被指"盯着美团、京东全年利润打"【网易,2025-07-04】[C级]
- **利润大幅下滑**：2026财年Q2经调整EBITA同比下降78%，经营利润53.65亿元，比2025财年同期少了近300亿【网易,2025-12-01】[C级]
- **竞争加剧**：电商"四巨头"竞争被称为"神仙打架"，拼多多成交总额5.2万亿元(增速30%)，抖音电商3.5万亿元与京东持平【百家号,2025-02-24】[C级]
- **商业模式冲击**：拼多多"社交裂变+算法推荐"削弱阿里依赖搜索购物的传统模式；抖音"货找人"模式改变用户购物路径【网易,2025-07-04】[C级]
- **"中年危机"**：网友调侃"阿里和京东就像中年大叔，拼多多和抖音才是活力满满的年轻选手"【百家号,2025-02-24】[C级]

**分析师观点**：2026财年中期利润表现不佳，近300亿的利润差别让市场担忧26财年盈利能力【网易,2025-12-01】[C级]

### 2.3 新业务巨额投入拖累盈利，现金流承压

**核心论点**：AI、云业务、淘宝闪购等新业务需要持续巨额投入，短期难以盈利，拖累整体财务表现。

**关键证据**：
- **资本开支激增**：2025年Q2资本开支386亿元，同比增长220%，创历史新高【网易,2025-09-01】[C级]
- **自由现金流净流出**：Q2财报显示自由现金流净流出218.4亿元，虽然现金储备仍有5856亿元，但流出趋势明显【网易,2025-12-01】[C级]
- **闪购UE亏损**：高盛预计阿里单笔闪购订单亏损将在12月季度收窄，但目前对比7/8月已降低一半，仍有亏损【网易,2025-12-01】[B级]
- **云业务投入**：未来三年计划投入3800亿元用于云和AI建设，短期难以产生对等回报【搜狐,2026-02-15】[C级]
- **研发投入担忧**：从通义千问到夸克AI、千问APP，AI落地过程中可能存在重复、无效研发投入，折旧摊销将影响未来利润【网易,2025-12-01】[C级]

**市场担忧**：分析师指出"阿里云对集团财务的贡献还远远不够"，2026财年中报云业务仅占合并经调整EBITA利润约13.7%【网易,2025-12-01】[C级]

### 2.4 监管风险持续，反垄断压力加大

**核心论点**：平台经济监管持续，反垄断合规要求提高，限制公司定价和竞争策略。

**关键证据**：
- **2026年2月约谈**：市场监管总局约谈阿里巴巴、抖音、百度、腾讯、京东、美团、淘宝闪购等7家平台企业，要求规范促销推广行为，杜绝"内卷式"竞争【IT Tiger,2026-02-14】[C级]
- **《互联网平台反垄断合规指引》**：2026年2月13日发布，共五章38条，明确8个场景新型垄断风险，包括"算法共谋"、"全网最低价"、"二选一"等【IT Tiger,2026-02-14】[C级]
- **"全网最低价"被点名**：要求平台内经营者在该平台销售商品价格不得高于其他竞争性平台，若在其他平台降价须在该平台降至相同或更低水平，被认定可能构成滥用市场支配地位【IT Tiger,2026-02-14】[C级]
- **算法监管**：要求对计价算法、推荐系统、排序逻辑等核心算法模型进行定向筛查和动态监测，确保"规则公平、算法向善"【IT Tiger,2026-02-14】[C级]
- **历史处罚**：2021年市监总局对阿里巴巴滥用市场支配地位行为作出处罚，罚款182.28亿元【绍耕,2026-02-10】[D级]

**学者观点**：对外经贸大学许可教授指出，当前外卖行业美团、阿里巴巴合计市场份额超80%，可推定具有共同市场支配地位【绍耕,2026-02-10】[D级]

## 三、市场关键争议点

### 3.1 电商份额能否守住30%关键关口？

**多方观点**：
- **招商证券**：阿里组织优化及用户为先战略调整驱动电商主业复苏，2024年上半年GMV增速恢复至接近实物网上零售额，市场份额趋稳。预计到2026年阿里增速将与电商行业持平、市占率将逐步稳定【新浪财经,2024-09-10】[B级]
- **阿里策略**：通过88VIP会员(超5000万，同比双位数增长)、全站推广告工具、淘工厂低价策略、闪购即时零售等方式巩固份额【阿里巴巴财报,2025-05-14】[A级]

**空方观点**：
- **市场数据**：阿里电商份额从80%跌至33%，趋势持续恶化。拼多多19%、抖音18%快速追赶【网易,2025-07-04】[C级]
- **用户行为变化**：年轻人更倾向抖音购物(边玩边买)、拼多多比价，传统搜索购物模式吸引力下降【百家号,2025-02-24】[C级]

**验证节点**：
- **季度GMV增速**：能否恢复至双位数增长
- **市场份额数据**：第三方机构季度份额报告
- **用户留存率**：高价值用户(年消费超1万元)留存率是否保持98%以上
- **88VIP会员增长**：能否持续双位数增长并提升客单价

### 3.2 云业务能否持续高增长？

**多方观点**：
- **CEO指引**：吴泳铭此前承诺"2025财年下半年商业化营收重返双位数增长"，Q2已实现34%增长【极客公园,2024-05-15】[B级]
- **AI驱动**：AI相关收入连续多个季度三位数增长，外部商业化收入中AI贡献超20%，带动计算、存储需求上升【网易,2025-09-01】[C级]
- **市场份额**：阿里云在中国AI云市场份额36%，处于领先地位【搜狐,2026-02-11】[C级]

**空方观点**：
- **竞争加剧**：华为云、腾讯云、百度智能云在AI领域激烈竞争，价格战持续
- **投入产出比**：3800亿投入能否产生相应回报？AI收入基数小，对整体利润贡献有限(13.7%)【网易,2025-12-01】[C级]
- **增速可持续性**：当前高增长部分源于低基数和AI热点，能否持续存疑

**验证节点**：
- **季度增速**：云收入能否连续8个季度保持20%+增长
- **AI收入占比**：外部商业化收入中AI贡献能否提升至30%+
- **利润率**：经调整EBITA利润率能否提升至20%+
- **市场份额**：能否保持35%+市场份额

### 3.3 AI能否转化为实际收入和利润？

**多方观点**：
- **三位数增长**：AI相关收入连续多个季度三位数增长，外部商业化收入中AI贡献超20%【网易,2025-09-01】[C级]
- **客户基础**：通义通过阿里云服务企业超9万家、通过钉钉服务企业超220万家，落地PC、手机、汽车、航空等多领域【极客公园,2024-05-15】[C级]
- **资本开支回报**：1000亿投入已带来显著收入增长，Q2资本开支386亿创历史新高，管理层信心十足【网易,2025-09-01】[C级]

**空方观点**：
- **基数小效应**：AI收入三位数增长主要因基数小，绝对金额有限
- **To C盈利困境**：千问APP等To C产品尚未跑出好的盈利模型，持续烧钱买量【网易,2025-12-01】[C级]
- **重复投入**：从通义千问到夸克AI、千问APP，可能存在重复研发，折旧摊销将影响未来利润【网易,2025-12-01】[C级]
- **"买预期"阶段**：市场对阿里的乐观还是在"买预期"，能否兑现存在不确定性【网易,2025-12-01】[C级]

**验证节点**：
- **AI收入绝对值**：AI相关收入占云收入比重能否提升至30%+
- **AI利润贡献**：AI业务能否实现盈利或至少盈亏平衡
- **To C产品数据**：千问APP月活用户、付费用户、ARPU值增长
- **投资回报率**：每投入1元AI资本开支带来的收入增长
