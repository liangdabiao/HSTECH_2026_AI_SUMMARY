**分析师**: Claude Code Stock Research Agent

## 执行摘要

阿里巴巴集团已成功转型为"AI+云"驱动的科技消费双引擎企业。2025财年集团总收入达**9,963.47亿元人民币**(约1,376亿美元)，同比增长6%；净利润1,259.76亿元，同比大增77%【来源A】。

**核心发现**:
- **中国商业**(淘天集团)仍是营收和利润支柱，占营收45.2%，但增速放缓至3%
- **云智能业务**成为新增长引擎，2025财年收入1,180.28亿元(+11%)，AI相关收入连续7个季度三位数增长
- **国际数字商业**增长最快，收入1,323亿元(+29%)，主要来自速卖通和Trendyol
- **本地生活**亏损持续收窄，高德已实现单季盈利
- **菜鸟物流**受业务调整影响，增速显著放缓
- **核心利润引擎**: 淘天集团CMR收入 + 阿里云EBITA利润

## 二、中国商业(淘天集团)深度分析

### 2.1 业务构成与收入

**淘天集团** = 淘宝 + 天猫 + 1688(内贸批发) + 飞猪(已整合至中国电商集团)

**2025财年业绩**:
- 总收入: **4,498.27亿元**，同比增长3%
- 客户管理收入(CMR): 同比增长6%【来源A】
- 88VIP会员数: 超**5,000万**，高质量消费群体【来源A】
- 经调整EBITA: 约**488亿元**(Q1数据)，利润率约45%

**Q4 FY25(2025年1-3月)亮点**:
- 收入1,013.69亿元，同比增长9%，创六个季度新高【来源B】
- 客户管理收入(CMR)同比增长12%

### 2.2 收入模式拆解

淘天集团收入主要来自三大来源:

1. **客户管理收入(CMR)**: 占比约70-75%
   - 广告收入: 通过阿里妈妈平台收取商家营销费用
   - 佣金收入: 天猫平台按销售额收取2-5%佣金【来源E】

2. **直营及其他**: 占比约20-25%
   - 天猫超市、盒马(已部分剥离)、高鑫零售(已出售)

3. **中国批发商业(1688)**: 占比约5%
   - B2B平台会员费、增值服务费

### 2.3 定价能力分析

**Take Rate(货币化率)现状**:

1. **技术服务费**:
   - 自2024年9月1日起，对淘宝商家收取**0.6%基础软件服务费**【来源C】
   - 这是淘宝历史上首次征收交易佣金，标志定价能力提升

2. **广告费率**:
   - 历史广告货币化率约**2.82%**(2024年)【来源E】
   - 新工具"全站推广"渗透率持续提升，拉动广告收入【来源D】

3. **佣金率**:
   - 天猫不同品类佣金率: **2-5%**【来源E】
   - 历史佣金货币化率约**1.1%**

**Take Rate提升趋势**:
- Q3 FY25: CMR增速(9%) > GMV增速(高个位数)，货币化率拐点已现【来源D】
- 驱动因素: 0.6%技术服务费 + 全站推广渗透 + 直播等新形式商业化深化

### 2.4 核心利润引擎评估

**淘天集团是阿里巴巴最核心的利润引擎**:

- **EBITA利润率**: 约45%(Q2 FY25)【来源F】，远超其他业务
- **EBITA贡献**: 约488亿元/季度，占集团EBITA总量的70%以上
- **现金流**: 稳定的经营现金流，支持其他业务投入

**2025财年战略转变**:
- 从"价格力"导向转向"GMV+体验"导向
- 优化"仅退款"政策，降低对优质商家的干预【来源C】
- 88VIP会员权益强化，提升高价值用户留存

## 三、云智能集团(阿里云)深度分析

### 3.1 收入与增长

**2025财年业绩**:
- 收入: **1,180.28亿元**，同比增长11%【来源G】
- 经调整EBITA: **105.56亿元**，同比增长72%【来源G】
- EBITA利润率: 从7.2%提升至9.9%

**季度加速趋势**:
- Q1 FY25: 265.49亿元(+6%)
- Q2 FY25: 296.1亿元(+7%)
- Q3 FY25: 317.42亿元(+13%)
- Q4 FY25: 301.27亿元(+18%)【来源G】
- **Q1 FY26: 398.2亿元(+34%)，增速创三年新高**【来源G】

### 3.2 AI驱动的增长引擎

**AI相关产品收入**:
- 连续**9个季度**实现三位数(100%+)同比增长(截至Q1 FY26)【来源G】
- AI收入占外部收入比重超**20%**
- 模型平台"百炼"付费用户季度环比增长超**200%**【来源G】

**AI基础设施投入**:
- 宣布未来三年投入**3,800亿元**建设云和AI硬件基础设施【来源A】
- 单季度CAPEX达386亿元(2025年Q2)，创历史新高【来源H】

### 3.3 市场地位与定价能力

**市场份额**:
- 中国AI云市场份额:**35.8%**(第一)，超过第2-4名总和【来源G】
- 服务约**63%**的中国A股上市公司【来源A】
- 为全球34个地区提供云计算服务【来源A】

**定价能力分析**:
1. **2024年2月大幅降价**:
   - 平均降价幅度超**20%**，最高降幅达**55%**【来源G】
   - 涉及100多款产品、500多个产品规格

2. **降价驱动用量增长**:
   - 公共云收入双位数增长
   - AI相关产品采用量大幅提升

3. **未来提价空间**:
   - 随着AI需求爆发和算力紧缺，云服务价格有望回升
   - 高附加值AI产品(如大模型服务)定价能力强

### 3.4 竞争优势

**全栈AI能力**:
- 基础设施: 磐久AI服务器、HPN 8.0网络
- 模型层: 通义千问Qwen系列，开源模型全球下载量超**3亿次**【来源A】
- 平台层: 百炼大模型平台
- 应用层: 夸克AI、钉钉AI、高德AI等

**生态协同**:
- 与淘宝、天猫、饿了么等阿里系应用深度整合
- 为企业提供"AI+云+应用"一体化解决方案

## 五、菜鸟物流深度分析

### 5.1 收入与利润

**2025财年业绩**:
- 收入: **1,012.72亿元**，同比增长**2%**【来源A】
- Q3 FY25收入: 282.41亿元，同比下降**1%**【来源A】
- Q3 FY25经调整EBITA: 2.35亿元，同比下降**76%**【来源A】

### 5.2 业务调整影响

**组织架构调整**(2025年初):
- 服务速卖通的400多人团队转入阿里电商事业群【来源A】
- 国内电商供应链解决方案和电子面单团队移交淘天集团【来源A】
- 菜鸟专注物流运营和服务职能，转型为"纯粹物流公司"

**调整影响**:
- 收入增速大幅下滑(2024财年+28% → 2025财年+2%)
- EBITA利润率从3.4%骤降至0.8%(Q3 FY25)【来源A】
- 阿里巴巴称: "电商业务承担部分物流平台职责"

### 5.3 核心能力与竞争优势

**全球物流网络**:
- 全球最大的跨境电商物流公司【来源A】
- "全球5日达"、"全球10日达"等服务标杆
- 海外本地快递网络布局(欧洲、北美、拉美)

**供应链优势**:
- 与淘天、速卖通深度协同
- 提升履约体验，缩短平均配送时长【来源D】

### 5.4 未来展望

**短期阵痛**:
- 收入和利润率受业务调整影响
- 战略定位重塑中

**长期价值**:
- 聚焦核心物流能力，提升服务外部客户能力
- 全球智能物流网络仍是稀缺资产
- 有望在调整后实现更高质量增长

## 十五、结论

### 15.1 业务质量评估

**优秀**:
- 淘天集团: 强大的现金流引擎，定价能力提升
- 阿里云: AI驱动下增长加速，利润率提升
- 国际商业: 高速增长，长期潜力大

**良好**:
- 本地生活: 亏损收窄，高德已盈利
- 战略创新业务: 钉钉、夸克快速增长

**改善中**:
- 菜鸟: 业务调整阵痛，长期价值仍存
- 大文娱: 亏损收窄，但非核心

### 15.2 战略转型评估

阿里巴巴已从"电商为主"成功转型为"AI+云+消费"双引擎驱动:

1. **核心现金牛**: 淘天集团稳定输出利润
2. **第二增长曲线**: 阿里云AI驱动加速增长
3. **长期潜力**: 国际商业规模化扩张
4. **生态协同**: 多业务AI化升级

### 15.3 核心观点

阿里巴巴的业务组合呈现**"1+1+N"**格局:
- **1个现金牛**: 淘天集团
- **1个增长引擎**: 阿里云(AI驱动)
- **N个潜力业务**: 国际商业、本地生活、创新业务

**关键催化剂**:
1. AI需求持续超预期 → 阿里云加速
2. 国际业务盈亏平衡 → 利润释放
3. 即时零售UE改善 → 新增长点
4. Take Rate提升 → 淘天利润增长

**关键风险**:
1. 电商竞争加剧 → 淘天份额流失
2. AI需求不及预期 → 云业务降速
3. 即时零售投入失控 → 现金流恶化
4. 地缘政治风险 → 国际业务受阻
