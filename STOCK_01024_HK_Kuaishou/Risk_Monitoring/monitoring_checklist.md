## 一、核心监控指标（按重要性排序）

### Tier 1: 最关键（必须监控，每季度）

#### 指标1: DAU增速（日活跃用户增长）

| 项目 | 内容 |
|------|------|
| **重要性** | ⭐⭐⭐⭐⭐ 核心指标 |
| **数据来源** | 公司财报 |
| **监控频率** | 每季度 |
| **预期值** | >5% |
| **警戒线** | <3%（连续2个季度） |
| **当前值** | +5.2%（2024年Q1） |
| **趋势** | 增速放缓 |
| **触发操作** | 若<3%，重新评估投资逻辑 |

**监控要点**:
- DAU是快手的生命线，直接决定广告库存和电商GMV
- DAU增速放缓至5%以下，说明用户增长见顶
- 若DAU下滑，需警惕竞争加剧或用户流失

#### 指标2: 电商GMV增速

| 项目 | 内容 |
|------|------|
| **重要性** | ⭐⭐⭐⭐⭐ 核心指标 |
| **数据来源** | 公司财报 |

| **预期值** | >15% |
| **警戒线** | <10%（连续2个季度） |
| **当前值** | +28.2%（2024年Q1），+15.1%（2024年Q3） |
| **趋势** | 增速放缓但仍高于行业 |
| **触发操作** | 若<10%，重新评估成长逻辑 |

**监控要点**:
- 电商GMV是快手第二增长曲线，增长速度决定估值
- 增速降至10%以下，说明电商增长见顶
- 需关注GMV结构（直播电商 vs 货架电商）

#### 指标3: 货币化率

| 项目 | 内容 |
|------|------|
| **重要性** | ⭐⭐⭐⭐⭐ 核心指标 |
| **数据来源** | 公司财报（计算） |

| **预期值** | >1.3%，持续提升 |
| **警戒线** | <1.2%或连续2季度不提升 |

**监控要点**:
- 货币化率提升是利润增长的关键驱动力
- 货币化率从1.2%提升至1.5%，可提升25%利润
- 需关注提升路径（泛货架占比、品牌商家、品类优化）

## 三、定期重新评估

### 3.1 季度评估（财报后）

**评估内容**:
1. 业绩是否符合预期（收入、利润、GMV）
2. 用户数据是否健康（DAU、时长、DAU/MAU）
3. 业务趋势是否改善（货币化率、广告增速、电商结构）
4. 竞争格局是否变化（抖音、视频号份额）
5. 估值是否合理（PE是否修复至15-18倍）

**操作决策**:
- 若符合或超预期 → 继续持有或加仓
- 若略低于预期（<10%） → 观察下一季度
- 若明显低于预期（>10%） → 重新评估，考虑减仓

### 3.2 半年评估（年中、年末）

**评估内容**:
1. 中期竞争格局（抖音、快手、视频号份额变化）
2. 中期增长趋势（收入、GMV增速是否稳定）
3. 中期护城河（用户粘性、私域流量是否保持）
4. 中期估值（DCF模型更新，安全边际是否足够）

**操作决策**:
- 若趋势向好 → 继续持有
- 若趋势恶化 → 重新评估投资逻辑，考虑清仓

### 3.3 年度评估（完整财年后）

**评估内容**:
1. 年度业绩回顾（收入、利润、GMV、DAU）
2. 年度目标达成率（是否达到年初预期）
3. 下一年度展望（管理层指引是否合理）
4. 长期投资逻辑是否变化（竞争、监管、行业）

**操作决策**:
- 若逻辑未变，继续持有
- 若逻辑恶化，清仓退出
- 若逻辑改善，加仓持有

## 四、关键风险监控

### 风险1: 竞争风险（抖音碾压）

**监控指标**:
- 市场份额（按用户时长）
- GMV份额
- 广告收入份额

**警戒线**:
- 份额下滑至<20%（当前26.5%）
- 连续2个季度份额下滑

### 风险2: 监管风险（内容合规）

**监控指标**:
- 监管处罚频率
- 直播收入增速
- 内容审核投入

**警戒线**:
- 出现重大监管事件（罚款、整改、下架）
- 直播收入连续下滑

### 风险3: 增长风险（用户增长见顶）

**监控指标**:
- DAU增速
- MAU增速
- 新用户获取成本

**警戒线**:
- DAU增速<3%（连续2个季度）
- DAU绝对值下滑

### 风险4: 盈利风险（利润高增长不可持续）

**监控指标**:
- 毛利率
- 费用率
- 利润增速

**警戒线**:
- 毛利率下滑至<50%
- 利润增速<30%（连续2个季度）

### 风险5: 头部主播风险（超头依赖）

**监控指标**:
- 头部主播GMV贡献
- 头部主播稳定性
- 中腰部主播成长

**警戒线**:
- 辛巴等超头出走或被封
- 头部主播GMV贡献>40%

## 七、紧急情况处理

### 情况1: 股价单日大跌>10%

**处理步骤**:
1. 查找下跌原因（新闻、公告）
2. 评估影响是否实质性
3. 若是情绪性波动，可考虑加仓
4. 若是实质性利空，考虑止损

### 情况2: 公司发布重大利空

**处理步骤**:
1. 仔细阅读公告，评估影响
2. 对比历史类似事件
3. 重新评估投资逻辑
4. 决定是否止损

### 情况3: 行业出现重大政策变化

**处理步骤**:
1. 评估政策对行业和公司的影响
2. 对比同行反应
3. 重新评估长期投资逻辑
4. 决定是否调整仓位
