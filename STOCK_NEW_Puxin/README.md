## 双减政策影响

### K12业务剥离

**时间**: 2021年12月31日

**原因**: 国家"双减"政策禁止K12学科培训

**影响**:
- K12业务占收入59%
- 上市主体失去主要收入来源
- 留学业务同比下降16.8%

---

## 破产原因分析

### 1. 双减政策直接冲击

- K12培训业务被禁止
- 主要收入来源被切断
- 业务转型困难

### 2. 疯狂并购导致债务危机

- 持续亏损及负债状态
- 大规模收购消耗资金
- 整合不达预期

### 3. 疫情双重打击

- 出国留学人数减少
- 留学业务收入下降16.8%
- 线下业务受疫情影响

### 4. 转型尝试失败

- 2022年2月宣布评估进入快餐业务
- 疫情下餐饮行业惨淡
- 转型设想破灭

## 教训与启示

### 对创业者

1. **并购风险**: 疯狂并购可能导致消化不良
2. **债务风险**: 高负债模式在危机时脆弱
3. **政策风险**: 教育行业政策敏感度极高
4. **现金流**: 现金流比利润更重要

### 对投资者

1. **并购模式风险**: 收购驱动增长难以持续
2. **政策研究**: 必须深入研究行业政策
3. **财务健康**: 关注负债率和现金流
4. **退市风险**: 中概股退市风险高

### 对行业

1. **K12培训**: 整个行业被政策终结
2. **转型困难**: 教育机构转型极其困难
3. **整合风险**: 并购整合难度大

## 研究结论

朴新教育是"双减"政策后首家破产清算的大型教育集团，其失败标志着疯狂并购模式在教育行业的终结。

**核心问题**:
1. 疯狂并购导致债务危机
2. 双减政策直接打击主要业务
3. 疫情加剧业务困难
4. 转型尝试失败

**历史价值**: 作为"双减"影响下教育行业破产的典型案例
