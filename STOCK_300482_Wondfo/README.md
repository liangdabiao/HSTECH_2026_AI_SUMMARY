   - PCR检测、基因测序等
5. **病理诊断**（新兴业务）:
   - AI辅助诊断等

**用户/客户**:
- 医院: 各级医院检验科
- 基层医疗: 社区卫生服务中心、乡镇卫生院
- 海外市场: 150多个国家和地区

**行业地位**:
- 中国POCT（即时检测）龙头
- 慢性病检测领域领先
- IVD行业领军企业

**关键里程碑**:
- 1992年: 公司成立

- 2022年: 营收56.81亿元（历史高峰）

- 2025年: 海外收入占比首次超过国内

### Phase 2: 行业周期分析

**行业阶段**: IVD行业深度调整期，后疫情时代业绩回落

**中国IVD行业（2025年）**:
- 市场规模: 约2000-2500亿元
- 增速: 深度调整，负增长
- 竞争格局:
  - 头部厂商: 万孚生物、迈瑞医疗、新产业等
  - 厂商众多，竞争激烈
- 关键趋势:
  - 疫情后新冠检测需求消失
  - 集采降价
  - 基层医疗预算紧缩
  - 出海成为新增长点

**POCT行业（2025年）**:
- 市场规模: 约300-400亿元
- 增速: 调整期，增速放缓
- 竞争格局:
  - 头部厂商: 万孚生物、诺唯赞等
  - 产品同质化严重
- 关键趋势:
  - 慢病检测需求稳定
  - 传染病检测回落
  - 出海加速

**分子诊断行业（2025年）**:
- 市场规模: 约200-300亿元
- 增速: 高增长，年增速15-20%
- 竞争格局:
  - 头部厂商: 华大基因、艾德生物等
  - 技术壁垒高
- 关键趋势:
  - 纳米孔测序技术兴起
  - AI赋能

**海外市场**:
- 发展中国家需求增长
- 性价比优势明显
- 中国IVD产品认可度提升

**监管环境**:
- 集采常态化
- 审评审批加速
- 出口认证支持

**周期判断**: IVD行业深度调整，新冠业务回落，企业依靠慢性病检测、海外拓展、创新业务突围。
