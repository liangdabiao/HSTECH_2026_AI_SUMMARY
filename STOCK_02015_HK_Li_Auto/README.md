**行业阶段**: 中国新能源汽车行业成长期，20万以上中高端市场成熟期

**中国新能源汽车行业（2025年）**:
- 市场规模: 约1000万辆/年
- 增速: 中速增长，年增速20-30%
- 竞争格局:
  - 头部厂商: 比亚迪、特斯拉、理想、蔚来、小鹏
  - 理想在20万以上SUV市场领先
- 关键趋势:
  - 渗透率持续提升
  - 价格战激烈
  - 智能化成为核心差异化
  - 出海加速

**中高端新能源SUV市场（20万以上）**:
- 市场规模: 约200万辆/年
- 增速: 中速增长
- 竞争格局:
  - 头部厂商: 理想、问界、蔚来、特斯拉Model Y
  - 理想份额约15%

**监管环境**:
- 新能源车补贴退坡
- 购置税减免延续
- 充电基础设施支持
- 智能驾驶监管加强
- 数据安全法规

**周期判断**: 新能源车行业成长期，中高端市场竞争激烈，理想增程技术有差异化优势。
