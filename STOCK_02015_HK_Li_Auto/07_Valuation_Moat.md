## 三、DCF估值

### 3.1 基准情景(概率40%)

**假设**:
- WACC: 10%
- 永续增长率: 2.5%
- 2026-2028年交付量CAGR: 25%
- 2028年净利率: 8%

**现金流预测**:
```
          2025E    2026E    2027E    2028E
营收(亿)  1,350   1,600   1,900   2,200
净利(亿)   30     120     160     200
FCF(亿)   -20      50     120     180
```

**DCF计算**:
```
现值(PV) = Σ(FCF / (1+WACC)^t) + 终值(TV)
PV = 50/1.1 + 120/1.1² + 180/1.1³ + TV
TV = 200 × (1+2.5%) / (10%-2.5%) = 2,733亿元
PV ≈ 1,900亿元 ≈ 270亿美元
目标价 = 270亿 / 10亿股 = 27美元
```

**目标价**: 27美元 (+42%)

### 3.2 牛市情景(概率30%)

**假设**:
- i6月销3万辆+,纯电转型超预期
- 2026-2028年交付量CAGR: 35%
- 2028年净利率: 10%

**目标价**: 35美元 (+84%)

### 3.3 熊市情景(概率30%)

**假设**:
- i6月销<1万辆,纯电转型失败
- 2026-2028年交付量CAGR: 5%
- 2028年净利率: 3%

**目标价**: 12美元 (-37%)

### 3.4 期望收益

**期望收益**:
```
E(R) = 40% × 42% + 30% × 84% + 30% × (-37%)
E(R) = 16.8% + 25.2% - 11.1%
E(R) = +30.9%
```

**年化收益**: +9% (3年持有期)

**风险收益比**: 1:1.5 (中等)

## 四、反向DCF

### 4.1 当前股价隐含增长

**当前股价**: 19美元

**隐含假设**:
```
假设WACC=10%,永续增长率=2.5%
反向推导2028年交付量:
19美元 = FCF / (WACC - g)
FCF ≈ 100亿元(约14亿美元)
对应2028年净利润约150亿元
对应交付量约60万辆
隐含2026-2028年CAGR约7%
```

**结论**: 当前股价隐含温和增长(CAGR 7%),低于市场预期(20%+)

### 4.2 敏感性分析

**关键变量敏感性**:

**最敏感变量**: 永续增长率、净利率

## 八、结论

### 8.1 护城河结论

**综合护城河**: 3/5 (中等偏强)

**优势**:
- ✅ 品牌认知度强("奶爸车"品类开创者)
- ✅ 规模效应显现(年交付50万辆)
- ✅ 成本控制能力强(毛利率20.5%)

**劣势**:
- ⚠️ 技术壁垒不高(增程式无专利壁垒)
- ⚠️ 智驾体验落后
- ⚠️ 充电网络规模不及竞品

**可持续性**:
- 短期(1-2年): 护城河稳定
- 中期(3-5年): 护城河可能被削弱
- 长期(5-10年): 依赖纯电转型成功

### 8.2 估值结论

**估值评级**: ⭐⭐⭐⭐ (偏低)

**理由**:
1. PS 1.0x处于历史低位
2. PE 18x低于同业平均
3. 现金储备占总市值40%,安全边际高
4. DCF目标价27美元,高于当前19美元

**期望收益**: +30.9% (3年)

### 8.3 投资建议

**护城河 + 估值**: 中等护城河 + 低估值 = 谨慎买入

**适合投资者**:
- ✅ 能承受2-3年转型期波动的投资者
- ✅ 相信纯电转型最终成功的投资者
- ❌ 不适合追求短期收益的投资者

**关键验证节点**: 2026年Q2-i6销量验证
