## 业绩亮点分析

### 1. 连续盈利能力

**盈利里程碑**:
- 2024年Q4: 首次GAAP扭亏为盈
- 2025年Q1-Q3: 连续4季度GAAP盈利
- 2025年全年: 迈向疫情后首次全年营收增长

**盈利驱动**:
- 营收双位数增长(+12%)
- 利润率持续提升
- 费用管控成效显著

### 2. 业务复苏态势

**核心RTE业务**:
- 呈现强劲复苏态势
- 迈向疫情后首次全年营收增长
- 为可持续盈利奠定基础

**分业务收入**:
- Agora业务: 1820万美元(+15.9%)
- 声网业务: 1.22亿元(+8.4%, 约1720万美元)

### 3. AI战略布局

**对话式AI产品全栈发布** (2025年Q3):
- **对话式AI开发套件R1-4G**: 基于8910芯片, 轻量化低功耗
- **对话式AI引擎2.0**: 优化"优雅打断", 新增声纹识别、电话外呼
- **对话式AI Studio**: 可视化编排, 1小时做出专属智能体
- **AI模型评测平台3.0**: 覆盖89模型、11地区、10语言

**业务意义**:
- 降低AI开发门槛
- 拓展潜在客户群
- 预计成为2026年增长重要动力

### 4. 规模化应用突破

**里程碑事件**:
- 年度服务分钟数首次突破1万亿分钟
- 标志实时互动进入规模化应用阶段

**技术验证**:
- 超级碗60万并发直播
- 毫秒级延迟稳定
