## 市场机遇

### 1. AI手机

**硬件升级**:
- 耗电量增加→电池容量提升
- 需要大面积FPC
- 高频高速传输优势

### 2. AR/VR

**市场增长**:
- 2029年全球出货量有望达0.38亿台
- 单机软板使用比例提升
- 轻量化集成化需求

### 3. 可穿戴设备

**需求特点**:
- 承载更多元器件
- 轻量化集成化
- 线路密度要求提高
