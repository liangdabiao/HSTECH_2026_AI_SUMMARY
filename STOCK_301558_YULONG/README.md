## 核心投资亮点

- 2025年业绩预告：净利润3650-5400万元（+154%至275.78%），预计扭亏为盈
- 跨境电商综合服务商，涵盖零售和物流
- 2024年营收17.51亿元，营收规模尚可

---

## 主要风险提示

- 🔴🔴 2024年归母净利润仅1436.88万元（-88.38%），业绩大幅下滑
- 🔴🔴 2024年扣非净利润-466.81万元（-103.72%），主营亏损
- 🔴🔴 2024年Q3单季度归母净利润-237.55万元（-109.21%），单季度亏损
- 🔴🔴 经营现金流净流入-4350.02万元（-124.30%），现金流恶化
- 🔴🔴 2024年ROE仅1.09%，盈利能力极弱
- 🔴🔴 新平台布局未达预期，经济效益尚未体现
- 🔴🔴 物流成本上涨、毛利率下降
