## ❌ 主要风险（Top 5）

### 1. 盈利质量风险 🔴

- 净利率仅8.5%，盈利能力薄弱
- 历史累计亏损超200亿元
- 降本增效空间可能有限
- 需持续验证盈利能力

### 2. 用户增长见顶风险 🟡

- MAU增速8%，DAU增速10%，增长放缓
- 获客成本高企
- 增量市场转为存量竞争
- 目标4亿MAU仍有空间

### 3. 竞争加剧风险 🟡

- 抖音、快手向中视频扩张
- 用户时长被分流
- 短视频平台算法优势明显
- B站用户时长优势缩小

### 4. 商业化受限风险 🟡

- 陈睿承诺"永不添加贴片广告"
- 广告加载率提升空间有限
- 商业化与社区氛围平衡困难
- 变现效率受限

### 5. UP主生态风险 🟡

- 激励政策调整导致收入下降
- 头部UP主停更现象
- 商业化与创作者利益平衡
- 310万UP主获得收入
