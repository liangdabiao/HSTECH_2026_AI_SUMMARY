# 公司历史 - 正保远程教育

**报告日期**: 2026年3月11日

---

## 第一章: 公司起源 (2000-2008)

### 1.1 创立背景

正保远程教育成立于2000年，由朱正东创立。

**创始人简介**:
- **姓名**: 朱正东
- **背景**: 教育行业从业者
- **愿景**: 通过互联网技术打破地域限制，为更多人提供优质教育资源

### 1.2 早期发展 (2000-2008)

在创立后的8年间，正保远程教育逐步发展:

**业务特点**:
- 纯在线教育模式
- 职业培训为主
- 会计培训起家

**关键里程碑**:
- 2000年: 公司成立
- 2001年: 中华会计网校上线
- 2005年: 建设工程教育网上线
- 2008年: 准备IPO

## 第二章: 纽交所IPO (2008)

### 2.1 IPO详情

**上市基本信息**:

| 项目 | 内容 |
|------|------|
| 上市日期 | 2008年7月30日 |
| 交易所 | 纽交所 (NYSE) |
| 股票代码 | DL |
| 发行价 | 7美元/ADS |
| 发行数量 | 875万ADS |
| 募资金额 | 约6,125万美元 |
| 承销商 | 未公开 |

**ADS结构**:
- 1份ADS = 4股普通股
- 总发行普通股: 3,500万股

### 2.2 IPO意义

正保远程教育是**2008年唯一一家**在美国纽交所上市的中国专业从事互联网远程教育的企业。

**行业地位**:
- 中国远程教育第一股
- 会计培训领域龙头
- 在线职业教育先驱

### 2.3 上市初期表现

| 时间点 | 股价 | 表现 |
|--------|------|------|
| 2008年7月 | 7美元 | IPO发行价 |
| 2008年末 | 约5-6美元 | 金融危机影响 |
| 2009年 | 约6-8美元 | 恢复性上涨 |
| 2010-2014年 | 约8-12美元 | 稳定增长 |
| 2015-2018年 | 约8-15美元 | 波动上升 |
| 2019-2020年 | 约9-10美元 | 相对稳定 |

## 第三章: 业务扩张 (2008-2015)

### 3.1 核心业务发展

#### 中华会计网校

**市场地位**:
- 中国会计远程教育领导者
- 市场占有率超过80%
- 品牌知名度第一

**主要产品**:
- 初级会计职称
- 中级会计职称
- 注册会计师(CPA)
- 税务师等

#### 建设工程教育网

**成立时间**: 2005年
**主要产品**:
- 一级建造师
- 二级建造师
- 造价工程师
- 监理工程师等

**市场地位**:
- 建筑类远程教育领先
- 市场占有率约40%

### 3.2 多元化布局

正保远程教育逐步扩展到多个职业教育领域:

| 网站名称 | 领域 | 上线时间 |
|---------|------|---------|
| 中华会计网校 | 会计 | 2001年 |
| 建设工程教育网 | 建筑 | 2005年 |
| 医学教育网 | 医学 | 2006年 |
| 法律教育网 | 法律 | 2007年 |
| 自考365网 | 自考 | 2008年 |
| 外语教育网 | 外语 | 2009年 |
| ITAT教育网 | IT | 2010年 |
| 中小学教育网 | K12 | 2011年 |

### 3.3 用户增长

**用户规模**:
- 2008年: 约50万人
- 2010年: 约100万人
- 2015年: 约200万人
- 2020年: 约320万人/年

## 第五章: 私有化与退市 (2020-2021)

### 5.1 私有化过程

#### 时间线

| 日期 | 事件 |
|------|------|
| 2020年6月8日 | 收到私有化要约 |
| 2020年12月1日 | 签署合并协议 |
| 2021年2月26日 | 股东大会审议 |
| 2021年上半年 | 完成私有化 |

#### 私有化要约详情

**初始要约** (2020年6月8日):
- 价格: 9.08美元/ADS
- 买方: 朱正东、殷保红及附属实体

**最终协议** (2020年12月):
- 价格: 9.80美元/ADS
- 普通股: 2.45美元/股
- 交易方式: 简式合并

### 5.2 私有化原因分析

1. **创始人控制需求**
   - 朱正东希望完全掌控公司
   - 避免公众公司压力

2. **估值合理**
   - 收购价接近市价
   - 对创始人成本可控

3. **运营灵活性**
   - 私营公司决策更快
   - 减少合规成本

4. **市场环境**
   - 中概股整体环境变化
   - 部分公司选择私有化
