## 📋 8阶段分析完整版

### Phase 1: 公司事实底座
**基本信息**: 雾芯科技成立于2018年，2021年纽交所上市，中国电子烟领导者
**核心业务**: 电子烟产品（悦刻品牌）、配件、零售
**产品矩阵**: 悦刻一代、五代、 infinity等系列
**客户**: 成年吸烟者
**创始人**: 汪莹（创始人、董事长兼CEO）

---

### Phase 2: 行业周期分析
**行业阶段**: 调整期，规范化发展期
**行业现状**: 电子烟新国标实施，行业规范化
**监管趋势**: 果味烟禁令、税率提高、销售限制
**竞争格局**: 悦刻领先，魔笛、雪加等竞争
**技术趋势**: 换弹式、一次性、HNB（加热不燃烧）

---

### Phase 3: 业务拆解分析

**业务本质**: 电子烟品牌商，悦刻品牌市占率领先

**业务构成（2025估算）**:
- **国内销售 (70%)**: 悦刻系列产品
- **国际业务 (30%)**: 海外市场快速扩张

**2025Q2业绩表现**:
- 营收: 8.8亿元（+40.3%，环比+8.9%）
- 净利润: 2.185亿元（+62%）
- Non-GAAP净利润: 2.912亿元
- 毛利率: 27.5%（+2.3pp）
- 运营利润: 3900万元（上年亏损）

**上半年业绩**:
- 营收: 16.88亿元（+43%）
- 净利润: 4.41亿元（+65%）

**Bull Case**:
- 高增长验证（营收+43%，净利+65%）
- 国际业务扩张成功
- 现金充裕（155亿元）
- 股东回报优厚（股息+回购）
