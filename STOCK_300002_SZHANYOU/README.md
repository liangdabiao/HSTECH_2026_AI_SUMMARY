### Phase 2: 行业周期分析

**行业阶段**: 游戏出海进入存量竞争，SLG赛道竞争加剧

**中国游戏出海行业（2025年）**:
- 市场规模: 约1500-2000亿元
- 增速: 存量竞争，增速放缓至5-10%
- 竞争格局:
  - 头部厂商: 腾讯、网易、莉莉丝、FunPlus、壳木游戏等
  - SLG赛道: 竞争激烈，优质竞品层出不穷
- 关键趋势:
  - SLG仍是出海主力品类
  - 混合玩法兴起（SLG+模拟经营、SLG+卡牌等）
  - 买量成本持续上升
  - 汇率波动对出海厂商影响大
  - 版号常态化，新游供给增加

**SLG游戏行业**:
- 市场规模: 约500-600亿元
- 竞争格局: 头部产品垄断，新品突围难
- 用户特征: 高ARPU、高付费率、长生命周期
- 关键成功要素: 长线运营能力、本地化运营、买量能力

**软件与信息技术服务行业**:
- AI+云服务: 市场快速增长，年增速20%+
- 物联网: 5G推动下快速发展
- 智能催收/电销: 金融科技需求旺盛

**监管环境**:
- 游戏版号常态化
- 网络游戏监管趋严
- 数据安全监管加强
- 跨境数据流动监管

**周期判断**: 游戏出海进入存量竞争期，SLG赛道竞争加剧，公司依靠长青产品和AI+云业务增长。
