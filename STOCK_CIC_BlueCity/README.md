## 退市原因分析

### 1. 二级市场表现不佳

- IPO发行价: 16美元/ADS
- 退市估值: 1.60美元/ADS
- 跌幅: -90%
- 市值大幅缩水

### 2. 商业化困难

- 海外用户价值挖掘不足
- 海外用户占比>40%，但收入仅占12%
- 广告增多影响用户体验
- 变现能力有限

### 3. 行业特殊性

- 政策风险
- 社会污名化
- 用户群体批评
- 发展难度比普通公司大

### 4. 盈利压力

- 持续亏损
- 营收增长乏力
- 获客成本高
- 变现效率低

## 投资教训

### 1. 小众市场风险

- LGBTQ市场规模有限
- 竞争加剧
- 增长天花板低

### 2. 上市时机选择

- 2020年上市，赶上疫情
- 市场环境不利
- 投资者信心不足

### 3. 商业化能力

- 用户基数大但变现难
- 海外拓展不成功
- 盈利模式单一

### 4. 估值合理性

- IPO估值过高
- 实际价值远低于发行价
- 私有化价格仅为发行价10%
