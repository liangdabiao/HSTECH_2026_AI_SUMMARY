  - 端到端大模型快速普及
  - "去高精地图"成为主流

  - Robotaxi商业化加速

- 渗透率过半，进入"存量替代+结构升级+全球化"阶段

- 增程/插混增速超纯电

**周期判断**: 智能驾驶成为竞争焦点，小鹏在智驾领域处于领先地位，"物理AI"叙事打开想象空间。
