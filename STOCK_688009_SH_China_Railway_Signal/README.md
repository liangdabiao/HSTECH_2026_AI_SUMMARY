### 核心投资逻辑（3句话）
1. **行业地位**：中国通号是高铁控制系统全球龙头，国内市占率60-70%，行业地位难以撼动。
2. **增长挑战**：轨道交通投资高峰期已过，公司营收连续两年下滑，需要依靠城轨更新、海外拓展、智能化升级寻找新增长点。
3. **估值安全**：当前PB约1.0x、PE约12-15x，处于历史低位，提供一定安全边际，但缺乏明显催化剂，适合"持有"而非"重仓买入"。

### 关键监控指标（季度跟踪）
1. **新签合同额**: 判断未来营收增长前瞻指标
2. **毛利率走势**: 判断盈利质量改善/恶化
3. **经营现金流净额**: 判断现金回款能力
4. **在手订单余额**: 判断短期业绩保障程度
5. **铁路固定资产投资完成额**: 判断行业景气度

### 退出信号
