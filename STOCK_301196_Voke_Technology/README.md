## 核心投资亮点

- **业绩高增长**：2024年营收+35.64%，净利润+31.05%
- **行业龙头**：中国精密注塑模具重点骨干企业
- **专精特新**：国家级"专精特新"小巨人企业
- **模塑一体化**：从模具到注塑件到健康产品的产业链协同
- **全球化布局**：生产基地覆盖中国、德国、马来西亚、墨西哥、越南
