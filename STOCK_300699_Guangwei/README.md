## 技术能力

### 1. 航天级碳纤维

**MJ系列**: 卫星用主流牌号
**M55J**: 高性能，单价显著高于M40J
**M40J**: 普及型卫星基板

### 2. 火箭应用

**应用部件**:
- 整流罩
- 发动机壳体
- 关键部件替代

**全箭体碳纤维化**:
- 我国首款全碳纤维复材液体火箭
- 箭体90%结构碳纤维化
- 较金属方案减重25-30%
