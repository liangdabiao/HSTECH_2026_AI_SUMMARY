## 核心结论

### ✅ 投资亮点

1. **汽车业务超预期兑现**
   - 18个月实现单季度盈利7亿元
   - Q3毛利率25.5%,超越特斯拉(16.3%)
   - 2025年交付目标35万辆,2026年55万辆

2. **生态壁垒全球领先**
   - AIoT平台连接10.4亿设备,全球最大
   - 全球MAU 7.42亿,开发者120万
   - 45%苹果用户拥有5件以上小米产品

3. **手机业务稳健增长**
   - 连续21个季度全球前三,份额13.6%
   - 高端化见效:4000-6000元价位段市占率18.9%
   - 2025年目标出货1.8亿台

4. **财务质量优异**
   - 2025年前三季度营收3,404亿元(+32.5%)
   - 经调整净利润328亿元(已超2024全年)
   - 现金储备2,367亿元

### ⚠️ 主要风险

1. **估值已部分反映乐观预期**
   - 当前PE 36倍,处于历史64%分位点
   - 市值1.88万亿港元,需持续高增长消化

2. **行业竞争加剧**
   - 华为回归手机市场,挤压高端空间
   - 新能源汽车价格战持续

3. **高管减持压力**
   - 副董事长林斌计划减持20亿美元,为期4年
