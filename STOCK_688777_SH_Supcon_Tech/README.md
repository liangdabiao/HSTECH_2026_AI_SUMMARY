## 🎯 核心投资论点

### 看好因素

1. **DCS 国内绝对龙头**: 市占率 40.4%，连续十三年第一
2. **SIS 三连冠**: 市占率 31.2%，持续领先
3. **工业 AI 转型**: TPT 大模型已规模化落地
4. **海外高增长**: 2024 年海外收入 +118%
5. **机器人业务**: 新兴增长曲线
6. **护城河深厚**: 转换成本高、品牌优势、成本优势

### 风险因素

1. **短期业绩承压**: 2025 年预计营收 -3.7%，净利润 -15%
2. **AI 商业化不确定性**: 转型节奏可能低于预期
3. **应收账款较高**: 应收账款/利润比率 295%
4. **估值波动**: 短期可能继续承压
