## 核心发现摘要

### ⚠️ 投资警示

1. **控制权变更**：2025年12月筹划控制权变更，原实控人曾本生转让股权给杭州拼便宜（徐意）
2. **业绩大幅下滑**：2024年亏损2370万元，2025年前三季度亏损2950万元
3. **产能扩张失误**：湖州嘉亨项目累计亏损1.39亿元，产能利用率仅32.76%
4. **二代接班失败**：曾焕彬接任董事长一年后公司业绩恶化，最终出让控制权
5. **行业竞争激烈**：化妆品代工行业价格战严重，毛利率持续下滑

### ✅ 潜在亮点

1. **优质客户基础**：服务科赴（原强生）、贝泰妮、上海家化等知名品牌
2. **全产业链能力**：具备研发、配制、灌装及塑料包装容器一体化服务能力
3. **新实控人资源**：拼便宜在供应链管理方面具有数字化优势和渠道资源
4. **行业长期增长**：中国化妆品市场预计2025年突破5790亿元
5. **国资背书**：浙江国资参与收购，具备资金和资源优势

### 🎯 投资信号评级

**综合评级：🔴🔴（两个红灯）- 观望/回避**

**评级理由**：
- 生意质量：🟡 一般（客户优质但竞争激烈、产能利用率低）
- 财务质量：🔴 差（连续亏损、现金流恶化、债务负担增加）
- 估值水平：🟡 中性（当前估值相对合理但业绩亏损）
- 治理风险：🔴 高（控制权变更、管理层不稳定）

### 💰 投资建议

**投资策略**：
- **保守型投资者**：回避，等待控制权变更完成、业绩明确改善信号
- **平衡型投资者**：观望，关注新实控人整合进展
- **激进型投资者**：谨慎关注，控制仓位不超过2%，设定严格止损
