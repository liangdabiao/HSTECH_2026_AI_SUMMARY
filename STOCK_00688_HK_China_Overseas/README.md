  - 核心城市率先复苏

  - REITs打通退出渠道
  - 运营能力差异化

**监管环境**:
- "房住不炒"基调不变
- 因城施策边际放松
- 融资支持"三个不低于"
- 保交楼政策持续推进

**周期判断**: 行业深度调整期，底部震荡，但央企龙头市场份额提升，中海逆势扩张。
