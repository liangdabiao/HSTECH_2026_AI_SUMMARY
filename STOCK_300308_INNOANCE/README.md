- 关键趋势:
  - 制造业复苏、设备更新需求释放
  - 国产替代深化
  - 智能制造、工业母机政策支持

- 市场规模: 约1万亿元

- 关键趋势:
  - 渗透率持续提升

  - 出海加速

**电梯行业（2025年）**:
- 市场规模: 约2000亿元

- 关键趋势:
  - 旧梯更新与后服务市场增长
  - 海外市场拓展

**监管环境**:
- 新型工业化政策支持
- 智能制造、工业母机政策加码
- 新能源汽车政策支持

**周期判断**: 工控行业弱复苏，新能源汽车高速增长，汇川技术凭借"工控龙头+新能源新贵"双轮驱动。

### Phase 6: 市场分歧

**Bull Case**:
1. **工控龙头**: 通用伺服、低压变频器市占率国内第一
2. **新能源汽车业务**: 营收90亿元（+50%），成为第一大收入来源

5. **研发投入**: 19.66亿元（+33.47%），研发人员占40%
6. **赴港上市**: 搭建"A+H"双资本平台，加速全球化
7. **国产替代**: 持续替代安川、三菱、西门子等外资品牌
8. **人形机器人**: 布局核心零部件，技术储备深化

**Bear Case**:
1. **增速放缓**: Q4增速较前三季度有所回落
2. **毛利率承压**: 新能源业务占比提升带来毛利率阶段性承压

6. **研发费用压力**: 研发与产能扩张带来费用压力
7. **海外拓展壁垒**: 地缘、认证与渠道壁垒
8. **高估值**: PE 50-60x，估值较高

**关键争议**: 增速能否维持？毛利率能否稳定？海外拓展能否持续？

**验证节点**:
1. 2025年年报: 全年业绩能否兑现
2. 2026年Q1: 制造业复苏能否持续
3. 2026年: 新能源汽车业务能否维持高增长
4. 2026年: 海外拓展能否加速

**共识 vs 分歧**:
