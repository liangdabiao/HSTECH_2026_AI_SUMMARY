## 💰 利润驱动因素分析

### 毛利率拆解

**整体毛利率**: 74%（2023年）

**分部毛利率**:
- 中低功率: 72%（稳定）
- 高功率: 78%（最高）
- 软件: 85%（持续提升）
- 其他: 60%（较低）

**毛利率驱动因素**:
```
毛利率 = 1 - (硬件成本 + 生产成本 + 其他成本) / 收入

硬件成本占比: 25-30%
├── 芯片/FPGA: 10-12%
├── 功率器件: 5-6%
├── PCB/连接器: 5-7%
└── 其他电子元器件: 5-8%

生产成本占比: 3-5%
├── 外协加工: 2-3%
├── 组装测试: 1-2%

其他成本占比: 2-3%
├── 包装物流: 1%
├── 质检: 1-2%
```

### 净利率拆解

**净利率**: 39%（2023年）

**费用率结构**:
```
净利率 = 毛利率 - 销售费用率 - 管理费用率 - 研发费用率 - 财务费用率

74% (毛利率)
- 5% (销售费用率)
- 4% (管理费用率)
- 18% (研发费用率)
- 0% (财务费用率，净利息收入)
- 8% (所得税等)
= 39% (净利率)
```

**费用率分析**:
- **销售费用率5%**: 较低（直销模式、客户黏性高）
- **管理费用率4%**: 较低（扁平化管理、效率高）
- **研发费用率18%**: 较高（技术驱动、持续投入）

## 📊 定价能力分析

### 定价策略

**中低功率系统**:
- 定价: 2-3万元/套
- 策略: 性价比优势
- 溢价能力: 中等（vs维宏股份溢价10-15%）

**高功率系统**:
- 定价: 5-8万元/套
- 策略: 技术溢价
- 溢价能力: 较高（vs外资产品折价20-30%）

**软件服务**:
- 定价: 0.5-2万元/年
- 策略: 价值定价
- 溢价能力: 高（毛利率85%+）

### 定价权评估

**优势**:
- 技术领先，定价主动权强
- 客户黏性高，涨价阻力小
- 竞争格局好，价格战缓和

**劣势**:
- 中低功率市场接近饱和，涨价空间有限
- 高功率市场外资竞争激烈，定价受限
- 下游设备厂商成本压力大

**结论**:
- 中低功率: 定价能力中等
- 高功率: 定价能力较强（随技术提升增强）
- 软件服务: 定价能力强
