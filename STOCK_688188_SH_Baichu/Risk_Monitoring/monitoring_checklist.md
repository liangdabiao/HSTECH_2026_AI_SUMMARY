## 🚨 风险预警信号

### 🔴 红色警报（立即减仓/止损）

**出现以下信号立即行动**:

1. **业绩恶化**
   - 连续2季度营收增速<10%
   - 连续2季度净利润增速<15%
   - 毛利率降至<68%

2. **竞争地位下降**
   - 市占率明显下降（年度下降>5ppt）
   - 丢失重要客户（前5大客户流失>2家）

3. **财务风险**
   - 经营现金流/净利润<0.7（连续2季度）
   - 应收账款周转天数>90天
   - 计提大额坏账准备

4. **治理问题**
   - 创始团队大幅减持（>5%）
   - 财务造假或重大违规
   - 实际控制人变更

5. **行业颠覆**
   - 新技术颠覆传统控制系统
   - 跨界巨头强势进入（如华为、阿里）

### 🟡 黄色预警（密切关注）

**出现以下信号密切关注**:

1. **增速放缓**
   - 单季度营收增速15-20%（预警区间）
   - 单季度净利润增速20-25%（预警区间）

2. **毛利率压力**
   - 单季度毛利率70-72%（预警区间）
   - 竞争对手降价

3. **客户流失**
   - 单一客户流失（非前5大）
   - 客户投诉增加

4. **高管变动**
   - 核心技术人员离职
   - 高管减持（<2%）

### 🟢 绿色安全（持续持有）

**以下情况持续持有**:

1. **业绩稳健**
   - 营收增速≥20%
   - 净利润增速≥25%
   - 毛利率≥73%

2. **竞争巩固**
   - 市占率维持或提升
   - 高功率突破顺利

3. **财务健康**
   - OCF/净利润≥1.0
   - 无重大财务风险

## 🎯 加仓/减仓触发点

### 加仓触发点

**触发条件**（满足任一即可）:

1. **估值修复**
   - 股价跌至230元以下（PE<30x）
   - 市场恐慌性杀跌（非基本面原因）

2. **业绩超预期**
   - 单季度净利润增速>30%
   - 高功率市占率快速提升

3. **重大突破**
   - 高功率产品重大技术突破
   - 海外市场重大订单

4. **行业催化剂**
   - 下游行业超预期复苏
   - 国产替代加速政策

**加仓策略**:
- 当前仓位3% → 加仓至5%
- 分批加仓，每跌5%加1%

### 减仓触发点

**触发条件**（满足任一即可）:

1. **估值泡沫**
   - 股价涨至350元以上（PE>45x）
   - 市场情绪过热

2. **业绩不及预期**
   - 连续2季度业绩miss
   - 全年业绩指引下调

3. **竞争恶化**
   - 价格战加剧
   - 市占率明显下降

4. **基本面恶化**
   - 毛利率持续下行至<70%
   - 大客户流失

**减仓策略**:
- 减半仓（5% → 2.5%）
- 触发红色警报则清仓
