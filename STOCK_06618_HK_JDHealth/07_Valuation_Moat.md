## 3. 绝对估值分析

### 3.1 DCF估值 (简化模型)

#### 3.1.1 关键假设

| 参数 | 假设 | 说明 |
|------|------|------|
| **当前收入** | 582亿元 | 2024年实际 |
| **收入CAGR (2025-2030)** | 15% | 瑞银预测 |
| **长期增长率** | 3% | 行业平均水平 |
| **净利率** | 8% → 10% | 逐步提升 |
| **WACC** | 9.2% | 华泰证券假设 |
| **永续增长率** | 2.5% | 保守假设 |

#### 3.1.2 现金流预测 (简化)

| 年份 | 收入 (亿元) | 净利率 | 净利润 (亿元) | 折现因子 | 现值 (亿元) |
|------|-----------|--------|--------------|---------|-----------|
| 2025 | 669 | 8% | 53.5 | 0.916 | 49.0 |
| 2026 | 770 | 9% | 69.3 | 0.839 | 58.1 |
| 2027 | 885 | 9% | 79.7 | 0.768 | 61.2 |
| 2028 | 1018 | 10% | 101.8 | 0.704 | 71.7 |
| 2029 | 1171 | 10% | 117.1 | 0.645 | 75.5 |
| **合计** | | | | | **315.5** |

**终值计算**:
- 2030年净利润: 1171 × 1.03 × 10% = 120.6亿元
- 终值: 120.6 / (0.092 - 0.025) = 1,800亿元
- 终值现值: 1,800 × 0.645 = 1,161亿元

**企业总价值**: 315.5 + 1,161 = 1,476.5亿元
**每股价值**: 1,476.5 / 32.7亿股 = 45港元

**DCF结论**: **合理价值 ~45港元** (与瑞银目标价一致)
