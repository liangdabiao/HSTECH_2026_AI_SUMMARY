## 1. 核心业务模式

### 1.1 商业定位
京东健康是中国领先的医疗健康平台之一，以"医药和健康产品供应链为核心、医疗服务为抓手"的**新型医疗健康服务企业**。

**核心定位**:
- **线上健康消费第一入口**
- **全场景、全流程健康解决方案提供商**
- **医检药闭环产品生态**

### 1.2 业务架构

京东健康的业务已从最初的医药零售，发展为具备提供"**商品+服务**"的综合能力：

│  │              │     │              │     │
│  │ • 自营药房   │     │ • 在线问诊   │     │

## 2. 收入构成

### 2.1 收入来源拆分

根据2024年财报数据：

| 收入类别 | 金额 | 增长率 | 占比 | 毛利率 |
|---------|------|--------|------|--------|
| **商品收入** | ~540亿元 | ~8% | ~93% | ~21% |
| - 医药及健康产品 | 主力 | - | 主力 | - |
| **服务收入** | ~44亿元 | +12.5% | ~7% | - |
| - 平台服务、数字营销 | 增长最快 | 双位数 | - | - |
| **总计** | **581.6亿元** | **8.7%** | 100% | **~22%** |

### 2.2 商品收入细分

- **处方药**: 快速增长，核心驱动力
- **OTC药品**: 稳定现金流
- **医疗器械**: 高毛利品类
- **保健品**: 常态化需求
- **健康服务**: 检测、护理等

## 3. 核心产品与服务

### 3.1 医药零售业务

#### 3.1.1 京东大药房 (自营)
- **行业地位**: 中国最大的在线零售药房 (按2019年收入计，市场份额29.8%)
- **SKU数量**: 超过4000万个SKUs
- **第三方商家**: 超过8万家 (同比增长超100%)

#### 3.1.2 全渠道布局 (O2O)
- **品牌**: "京东买药"即送服务
- **配送速度**: 平均28分钟，最快9分钟
- **覆盖城市**: 超200个城市
- **医保支付**: 已在北京、上海、广州、深圳、东莞等地上线

#### 3.1.3 新特药首发优势
2024年上半年首发多款重磅新药：
- 兴奇眼药 0.01%硫酸阿托品滴眼液
- Leqembi (卫材/百健) - 阿尔茨海默病药物
- 众多专科新药选择京东健康作为"新特药上线首站"

### 3.2 医疗健康服务

#### 3.2.1 在线诊疗
- **日均问诊量**: 超48万例
- **互联网医院**: 已实现超80%医生问诊单使用AI服务
- **AI营养师**: 服务满意度达91%

#### 3.2.2 家庭快检服务
- **服务数量**: 83个快检服务产品
- **服务内容**:
  - 呼吸道病毒和细菌感染检测
  - 护士居家护理
  - 居家采样服务

#### 3.2.3 专科诊疗
- 心脑血管疾病教育及康复平台
- 风湿骨外科等领域数字化服务
- 糖尿病、高血压等慢病管理

### 3.3 数字化与AI能力

#### 3.3.1 AI京医产品体系
- **发布时间**: 2025年1月
- **功能**: 智能诊疗辅助、科研支持
- **应用场景**: 线上多场景应用

#### 3.3.2 京医千询大模型
- **定位**: 国内医疗行业首个全面开源的垂类大模型
- **发布时间**: 2025年2月
- **能力**: 医疗知识问答、辅助诊疗

### 5.1 京东物流协同

京东健康的核心竞争优势之一是依托**京东集团的供应链基础设施**：

| 资源 | 规模 | 优势 |
|------|------|------|
| **药品仓库** | 11个专业药品仓库 | GSP认证 |

| **配送服务** | 当日达、次日达、30分钟、7×24小时 | 极速体验 |
| **全渠道覆盖** | 超200个城市 | 即时零售 |

### 5.2 供应链护城河

- **低温仓储**: 专业药品仓储能力
- **质量追溯**: 全流程质量管控
- **智能分单**: AI驱动的库存优化
- **DTP药房**: 专业特药配送能力

## 8. 业务质量评估

### 8.1 生意模式特征

✅ **优势**:
- 高频刚需 (医药健康)
- 强复购率 (慢病患者)
- 网络效应 (用户越多，医生越多)
- 规模效应 (采购成本下降)

⚠️ **挑战**:
- 监管不确定性 (网售处方药)
- 毛利率受限 (药品利润空间)
- 竞争加剧 (阿里健康等)

### 8.2 收入质量

- **经常性收入**: 慢病复购带来稳定现金流
- **季节性**: 相对平稳 (除疫情特殊时期)
- **客户粘性**: 高 (用户切换成本高)
- **定价权**: 中等 (受医保支付、集采影响)

## 9. 关键成功要素

### 9.1 已建立的优势

1. **供应链优势**: 京东物流网络难以复制
2. **用户信任**: 京东品牌背书
3. **技术能力**: AI医疗领先
4. **全渠道**: 线上线下融合
5. **资本实力**: 京东集团支持

### 9.2 待强化的领域

1. **医疗服务能力**: 相对传统医疗机构仍有差距
2. **线下布局**: 需要更多线下医疗机构
3. **国际化**: 尚未实质性出海
4. **创新药渠道**: 需要更多药企合作

## 10. 业务基础总结

### 10.1 投资亮点

| 要素 | 评级 | 说明 |
|------|------|------|
| **商业模式** | ⭐⭐⭐⭐ | 刚需高频、网络效应强 |
| **市场地位** | ⭐⭐⭐⭐⭐ | 行业龙头、市场份额领先 |
| **供应链** | ⭐⭐⭐⭐⭐ | 京东物流协同、难以复制 |
| **技术能力** | ⭐⭐⭐⭐ | AI医疗领先、数字化强 |
| **用户基础** | ⭐⭐⭐⭐ | 1.81亿用户、增长稳健 |

### 10.2 关注要点

1. **增长放缓**: 营收增速从高位降至个位数
2. **盈利压力**: 需要平衡用户获取与利润
3. **监管风险**: 网售处方药政策变化
4. **竞争格局**: 阿里健康等追赶者
