   - 医疗器械、保健品
   - 中药饮片
2. **在线医疗健康服务**（占收入约15%）:
   - 京东互联网医院
   - 在线问诊、家庭医生
   - 慢病管理、心理咨询
3. **消费医疗服务**（占收入约5%）:
   - 体检服务
   - 医美服务
   - 口腔护理
   - 疫苗接种

**用户/客户**:
- C端用户: 年度活跃用户数超2亿
- B端商户: 入驻商家超数万家
- 医生: 注册医生超20万
- 医疗机构: 合作医院超数万家

**行业地位**:
- 中国最大在线医疗健康平台之一
- 医药电商市场份额约25%（与阿里健康并列）
- B2C药品电商市场第一

**关键里程碑**:

- 2024年: 营收518亿元，净利润21亿元

**业务本质**: 中国领先的在线医疗健康平台，"医药+医疗"双轮驱动。

| 在线医疗 | ~78 | 15% | +15% | 高 |
| 消费医疗 | ~26 | 5% | +20% | 中 |
| **总计** | **518** | **100%** | **+9.0%** | **中** |

**2024年关键数据**:
- 营收: 518亿元（+9.0%）
- 净利润: 21亿元（+233%）
- 调整后净利润: 26亿元（+21%）
- 净利率: 5.0%（提升2.8pct）
- 年度活跃用户: 超2亿

**利润引擎分析**:
1. **医药零售稳健**: 收入约414亿元（+8%），京东大药房收入超400亿

3. **消费医疗扩张**: 收入约26亿元（+20%），体检医美快速增长
4. **毛利率提升**: 规模效应和供应链优化
5. **费用率下降**: 营销效率提升

**定价权**: ⭐⭐⭐ (3/5)

### Phase 6: 市场分歧

**Bull Case**:
1. **业绩增长**: 2024年营收518亿元（+9.0%），净利润21亿元（+233%）

3. **市场份额**: 医药电商市场份额25%，龙头地位
4. **用户规模**: 年度活跃用户超2亿
5. **供应链优势**: 依托京东物流，28城药品小时达
6. **在线医疗**: 问诊量超1亿人次
7. **政策支持**: 医保在线支付逐步放开

**Bear Case**:
1. **竞争加剧**: 阿里健康、平安健康竞争激烈
2. **监管风险**: 互联网医疗监管趋严

4. **估值偏高**: PS约5x，PE约50x
5. **变现效率**: 在线医疗变现效率待提升
6. **依赖京东**: 业务依赖京东集团生态

**关键争议**: 在线医疗能否实现规模化盈利？医保支付接入进度？竞争格局演变？

**验证节点**:

2. 2025年: 净利率能否维持5%+
3. 2025年: 在线医疗变现能否加速
4. 2025年: 医保支付接入进展
5. 2025年: 市场份额能否维持

**共识 vs 分歧**:
- 共识: 京东健康是医药电商龙头，盈利能力强，用户规模大
- 分歧: 在线医疗变现效率？医保支付接入？估值是否合理？
