- 🟢🟢 2024年营收209.83亿元（+3.02%），营收突破200亿大关
- 🟢🟢 2024年归母净利润35.56亿元（+5.87%），净利润超35亿
- 🟢🟢 毛利率51.02%，净利率约17%，盈利能力出色
- 🟢🟢 2025年Q1营收60.26亿元（+15.97%），增速回升
- 🟢🟢 2025年Q1净利润10.5亿元（+16.71%），恢复双位数增长
- 🟢🟢 全球眼科医疗网络布局，境内352家医院+229家门诊部
- 🟢🟢 境外163家眼科中心及诊所，国际化战略持续推进
- 🟢🟢 上市以来累计分红超70亿元，持续回报股东
