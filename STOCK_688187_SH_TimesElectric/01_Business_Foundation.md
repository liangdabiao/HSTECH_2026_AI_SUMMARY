## 🏢 公司基本情况

### 公司概况

**公司全称**: 株洲中车时代电气股份有限公司
**英文名称**: CRRC Times Electric Co., Ltd.
**股票代码**: 688187.SH (科创板)
**H股代码**: 3898.HK (香港联交所)
**成立时间**: 2005年9月28日
**上市时间**: 2021年9月7日 (科创板)
**总部地址**: 湖南省株洲市石峰区时代路
**法人代表**: 李东林
**注册资本**: 12.43亿元

**公司定位**:
- 中国轨道交通装备制造领军企业
- 中国功率半导体龙头企业
- 新能源汽车电驱系统核心供应商
- 工业变流产品综合解决方案提供商

### 发展历程

**2005年**: 公司成立
- 由南车株洲电力机车研究所有限公司联合其他投资者共同发起设立
- 专注于轨道交通牵引变流系统研发制造

**2006-2010年**: 技术积累期
- 突破高铁牵引变流系统核心技术
- 获得CRH380动车组牵引系统订单
- 市场份额快速提升

**2011-2015年**: 业务扩张期
- 收购英国Dynex Semiconductor，布局IGBT芯片
- 建设国内首条8英寸IGBT芯片生产线
- 产品线从轨道交通向功率半导体、新能源扩展

**2016-2020年**: 产业升级期
- 中国中车南车北车合并，成为中国中车核心子公司
- IGBT芯片实现国产化突破
- 布局新能源汽车电驱系统、SiC功率器件

**2021年**: 科创板上市
- 2021年9月7日在上海证券交易所科创板上市
- 募集资金约70亿元，用于功率半导体、新能源汽车等项目

**2022-2024年**: 创新发展期
- SiC芯片产线建成投产
- 新能源汽车电驱系统业务快速增长
- 海外市场拓展取得突破

- 直接持股比例: 约38%

## 🚄 核心产品与服务

### 业务板块

公司业务主要分为四大板块:

#### 1. 轨道交通业务 (收入占比约50-55%)

**主要产品**:
- **牵引变流系统**
  - 高铁动车组牵引变流器
  - 城市轨道交通牵引变流器
  - 机车牵引变流器
- **辅助变流系统**
  - 辅助电源装置
  - 蓄电池充电机
- **其他关键部件**
  - 牵引变压器
  - 网侧变流器
  - 制动电阻

**产品特点**:
- 技术含量高，进入壁垒极高
- 与整车安全性、可靠性直接相关
- 需要长期技术积累和认证
- 产品生命周期长 (通常10-15年)

**市场地位**:
- 高铁动车组牵引系统市场份额: 80%+
- 城市地铁牵引系统市场份额: 50-60%
- 机车牵引系统市场份额: 60-70%

**客户群体**:
- 中国中车各子公司 (中车四方、中车长客、中车唐山等)
- 地方地铁公司 (北京地铁、上海地铁、广州地铁等)
- 海外铁路运营商 (中车海外订单配套)

#### 2. 功率半导体业务 (收入占比约25-30%)

**主要产品**:
- **IGBT芯片/模块**
  - 低压IGBT (600V-1200V): 家电、工业变频
  - 中压IGBT (1700V-3300V): 新能源汽车、光伏逆变
  - 高压IGBT (4500V-6500V): 轨道交通、电网
- **SiC器件**
  - SiC MOSFET芯片
  - SiC SBD芯片
  - SiC功率模块
- **其他功率器件**
  - 晶闸管 (SCR)
  - 整流管
  - 智能功率模块 (IPM)

**产品特点**:
- 技术密集，研发投入巨大
- 产品认证周期长 (汽车级需2-3年)
- 产品迭代快 (摩尔定律)
- 产业链长 (设计-制造-封装-测试)

**技术能力**:
- 拥有6英寸、8英寸IGBT芯片生产线
- 掌握芯片设计、制造、封装测试全产业链技术
- SiC器件技术国内领先
- 年产IGBT芯片超过200万片

**应用领域**:
- 轨道交通 (自用+外销)
- 新能源汽车 (主驱逆变器、OBC、DC-DC)
- 新能源发电 (光伏逆变器、风电变流器)
- 工业变频 (高压变频器、中低压变频器)
- 白色家电 (变频空调、冰箱、洗衣机)

**市场地位**:
- 轨道交通用IGBT: 国内市场份额第一 (约60%)
- 新能源汽车用IGBT: 国内市场份额第二 (约15%)
- 工业变频用IGBT: 国内市场份额前三

**客户群体**:
- 轨道交通: 自用为主，外销给其他轨道交通装备厂商
- 新能源汽车: 广汽、长安、理想、小鹏等
- 工业领域: 汇川技术、英威腾、伟创等
- 家电领域: 美的、格力、海尔等

#### 3. 新能源汽车电驱系统业务 (收入占比约10-15%)

**主要产品**:
- **驱动电机系统**
  - 永磁同步电机
  - 异步感应电机
- **电机控制器**
  - 基于IGBT的控制器
  - 基于SiC的控制器 (800V平台)
- **集成电驱系统**
  - 三合一电驱系统 (电机+电控+减速器)
  - 多合一电动力总成

**产品特点**:
- 乘用车为主，商用车为辅
- 向高功率密度、高效率、集成化发展
- 800V高压平台是趋势
- SiC器件应用增加

**技术能力**:
- 掌握电机、电控核心算法
- 拥有完整的电磁仿真、热仿真、机械仿真能力
- 年产能超过50万套

**市场地位**:
- 新能源汽车电驱系统市场份额: 约5-8%
- 800V SiC电驱系统领先供应商之一

**客户群体**:
- 广汽集团 (埃安品牌)
- 长安汽车 (深蓝品牌)
- 理想汽车
- 小鹏汽车
- 其他新势力车企

#### 4. 工业变流业务 (收入占比约10-12%)

**主要产品**:
- **高压变频器**
  - 2.3kV、3.3kV、4.16kV等级
  - 功率等级: 200kW-10MW
- **中低压变频器**
  - 380V、690V等级
  - 功率等级: 0.4kW-3MW
- **特种电源**
  - UPS不间断电源
  - 充电桩电源模块
  - 电镀电源
  - 感应加热电源

**产品特点**:
- 工业应用，可靠性要求高
- 定制化程度高，非标产品多
- 服务要求高，需要快速响应
- 国产替代空间大

**应用领域**:
- 电力、冶金、化工、建材、矿山、水务等
- 市政基础设施
- 数据中心
- 充电设施

**市场地位**:
- 高压变频器市场份额: 约8-10%
- 中低压变频器市场份额: 约3-5%

**客户群体**:
- 工业企业 (国家电网、南方电网、各大钢铁厂等)
- 系统集成商
- OEM厂商

## 👥 客户分析

### 客户集中度

**前五大客户销售情况** (2021-2023年):

| 年份 | 前五大客户收入占比 | 第一大客户占比 |
|------|------------------|----------------|
| 2021年 | 72.5% | 48.2% |
| 2022年 | 71.8% | 46.5% |
| 2023年 | 70.2% | 45.1% |

**客户集中度分析**:
- 前五大客户收入占比约70%，集中度高
- 第一大客户 (中国中车) 占比约45%，存在明显依赖
- 近三年客户集中度略有下降，但依然处于高位
- 主要原因是轨道交通业务占比高，客户结构单一

**主要客户** (2023年):

| 客户名称 | 销售额(亿元) | 占比 | 业务类型 |
|----------|-------------|------|----------|
| 中国中车股份有限公司 | 81.2 | 45.1% | 轨道交通设备 |
| 广汽集团 | 12.6 | 7.0% | 电驱系统 |
| 长安汽车 | 9.0 | 5.0% | 电驱系统 |
| 理想汽车 | 7.2 | 4.0% | 电驱系统 |
| 国家电网 | 5.4 | 3.0% | 功率器件、变流器 |
| **前五大合计** | **115.4** | **70.2%** | - |

**客户集中度风险评估**:
- **高风险**: 第一大客户占比超过40%，存在单一客户依赖
- **定价权弱**: 对大客户议价能力弱，毛利率承压
- **业绩波动**: 大客户订单变化直接影响业绩
- **改善方向**: 提升IGBT外销占比，拓展新客户

### 客户粘性分析

**轨道交通客户**:
- 中国中车体系内客户粘性极强
- 牵引变流系统是关键安全部件，更换成本极高
- 供应商切换周期长 (5-10年)
- 客户忠诚度高，但定价权弱

**新能源汽车客户**:
- 车企供应商认证周期长 (2-3年)
- 一旦进入供应链，粘性较强
- 但车企为了供应链安全，通常有多家供应商
- 需要持续技术升级，否则存在被替换风险

**工业客户**:
- 客户分散，粘性相对较弱
- 价格敏感度高，容易被低价竞争对手替代
- 需要提供优质服务和技术支持

## 🔗 产业链位置与价值链

### 产业链位置

#### 轨道交通装备产业链

```
上游: 原材料供应商
  ├─ 钢铁、铝合金
  ├─ 铜材、稀土永磁材料
  └─ 电子元器件

中游: 装备制造商
  ├─ 中国中车 (整车)
  ├─ 时代电气 (牵引系统) ← 我们在这里
  └─ 其他关键部件供应商

下游: 运营服务商
  ├─ 国家铁路集团
  ├─ 地方地铁公司
  └─ 海外铁路运营商
```

**产业链位置**: 中游核心零部件供应商
- 在轨道交通装备产业链中处于关键位置
- 牵引变流系统是动车组、地铁的核心部件
- 价值量占整车成本约15-20%
- 技术壁垒高，议价能力较强

#### 功率半导体产业链

```
上游: 材料与设备
  ├─ 硅片、SiC晶片 (材料)
  ├─ 光刻机、刻蚀机 (设备)
  └─ 特种气体、靶材 (辅料)

中游: 芯片设计制造
  ├─ 芯片设计 (时代电气) ← 我们在这里
  ├─ 晶圆制造 (时代电气+代工厂)
  └─ 封装测试 (时代电气)

下游: 应用领域
  ├─ 轨道交通 (自用)
  ├─ 新能源汽车
  ├─ 新能源发电
  └─ 工业变频
```

**产业链位置**: 中游IDM模式 (设计+制造+封装)
- 拥有完整的IGBT产业链
- 在芯片设计环节技术领先
- 在制造环节拥有自己的产线
- 在封装测试环节能力完备

#### 新能源汽车产业链

```
上游: 原材料
  ├─ 硅钢、稀土
  ├─ IGBT/SiC芯片
  └─ 电子元器件

中游: 核心部件
  ├─ 电池系统
  ├─ 电驱系统 (时代电气) ← 我们在这里
  └─ 电控系统

下游: 整车制造
  ├─ 传统车企转型
  ├─ 新势力车企
  └─ 客户 (广汽、长安、理想等)
```

**产业链位置**: 中游核心部件供应商
- 电驱系统是新能源汽车核心部件
- 价值量占整车成本约10-15%
- 技术壁垒高，竞争激烈
- 对上游IGBT芯片有自主可控优势

### 价值链分析

#### 各业务板块价值链位置

**轨道交通业务**: 高价值链位置
- 占据核心技术环节
- 价值量占整车15-20%
- 毛利率高 (35-38%)
- 但对整车厂议价能力一般

**功率半导体业务**: 中高价值链位置
- 芯片设计是核心环节
- 价值量占终端产品5-10%
- 毛利率中高 (28-32%)
- 对下游客户有一定议价能力

**新能源汽车电驱业务**: 中等价值链位置
- 整车厂价值量占比10-15%
- 毛利率较低 (15-20%)
- 对整车厂议价能力弱
- 竞争激烈，盈利困难

**工业变流业务**: 中等价值链位置
- 价值量占工业设备5-8%
- 毛利率中等 (25-28%)
- 客户分散，议价能力一般

### 纵向一体化程度

**高度一体化**:
- 功率半导体: IDM模式，从芯片设计到封装测试全流程自主
- 轨道交通牵引系统: 从IGBT芯片到系统级产品全流程自主

**一体化优势**:
- 供应链安全可控
- 成本优势明显
- 技术迭代速度快
- 定制化能力强

**一体化劣势**:
- 资本开支巨大
- 运营管理复杂

## 🎯 战略规划与未来方向

### "十四五"战略规划 (2021-2025年)

**战略定位**:
- 成为世界一流的轨道交通和功率半导体企业
- 实现轨道交通装备和功率半导体双轮驱动

**战略目标** (2025年):
- 营业收入: 达到300亿元
- 功率半导体收入占比: 提升至35%
- 海外收入占比: 提升至10%
- 研发投入占比: 维持在12-15%
- ROE: 提升至15%以上

**三大战略举措**:

1. **技术领先战略**
   - 持续加大研发投入，保持技术领先
   - 重点突破SiC、GaN等第三代半导体
   - 布局第四代半导体 (超宽禁带半导体)
   - 建设国家级功率半导体技术创新中心

2. **产业扩张战略**
   - 扩大功率半导体产能 (IGBT、SiC)
   - 加快新能源汽车电驱业务扩张
   - 拓展海外市场，提升国际竞争力
   - 布局新兴应用领域 (储能、氢能等)

3. **数字化转型战略**
   - 推进智能制造和工业互联网
   - 建设数字化研发平台
   - 实现供应链数字化管理
   - 推动产品智能化升级

### 业务发展重点

#### 功率半导体业务
- **IGBT**: 扩大产能，提升市占率，目标是国内市场份额达到25%
- **SiC**: 快速放量，2025年收入占比达到10%，争取国内前三
- **新兴器件**: 布局GaN、氧化镓等第四代半导体

#### 新能源汽车电驱业务
- **800V平台**: 加快800V SiC电驱系统研发和量产
- **集成化**: 推动三合一、多合一电驱总成
- **客户拓展**: 获得更多车企定点，目标市场份额10%+

#### 轨道交通业务
- **存量更新**: 高铁、地铁车辆更新换代带来存量市场
- **智能化**: 发展智能牵引系统、预测性维护
- **海外市场**: 跟随中车出海，拓展海外市场

#### 工业变流业务
- **高压变频**: 巩固高压变频器市场地位
- **细分市场**: 拓展特种电源、充电桩电源等细分市场
- **国产替代**: 加速国产替代进程

## 📈 核心竞争力总结

### 技术优势
1. 轨道交通牵引变流系统国内领先，国际先进
2. IGBT功率半导体国内龙头，全产业链布局
3. SiC器件快速追赶，技术水平国内前三
4. 研发投入强度高，技术创新能力强

### 市场优势
1. 轨道交通牵引系统垄断地位 (市场份额60-70%)
2. IGBT在轨道交通领域绝对优势
3. 新能源汽车电驱系统快速崛起
4. 母公司中国中车强力支持

### 产业链优势
1. 功率半导体IDM模式，全产业链自主可控
2. 从器件到系统级的完整产品线
3. 纵向一体化带来成本优势和技术协同

### 品牌与资质优势
1. 央企背景，品牌信誉度高
2. 长期积累的行业经验和客户关系
3. 完善的资质认证体系

## ⚠️ 主要风险提示

1. **客户集中风险**: 前五大客户占比70%+，依赖中国中车
2. **技术迭代风险**: 半导体技术快速迭代，需要持续大规模研发投入
3. **竞争加剧风险**: 功率半导体、新能源汽车电驱领域竞争激烈
4. **行业周期风险**: 轨道交通投资受政府财政影响，存在周期性
5. **关联交易风险**: 与母公司中国中车关联交易占比高
