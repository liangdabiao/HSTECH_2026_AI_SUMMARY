## 📋 8阶段分析完整版

### Phase 1: 公司事实底座
**基本信息**: 高途（原跟谁学）成立于2014年，2020年纽交所上市，中国领先的在线教育公司
**核心业务**: K12在线教育、成人教育、职业教育、AI教育
**产品矩阵**: 高途课堂、高途学院、AI智能学习系统
**客户**: K12学生、成人学习者
**创始人**: 陈向东（创始人/董事长兼CEO），前新东方执行总裁

---

### Phase 2: 行业周期分析
**行业阶段**: 复苏期，转型期
**行业现状**: 双减政策后转型，职业教育、成人教育成重点
**监管趋势**: K12严格监管，职业教育政策支持
**竞争格局**: 新东方、好未来转型，竞争加剧
**技术趋势**: AI+教育成为主流

---

### Phase 3: 业务拆解分析

**业务本质**: 在线教育平台，K12转型职业/成人教育

**业务构成（2025Q1估算）**:
- **K12辅导 (40%)**: 非学科类辅导
- **成人教育 (30%)**: 考研、公考、证书培训
- **职业教育 (20%)**: IT、技能培训
- **其他 (10%)**: 教育硬件、AI产品

**2025Q1业绩表现**:
- 营收: 14.93亿元（+58%）
- 净利润: 1.24亿元
- Non-GAAP净利润: 1.4亿元（净利率9.2%）
- 营业利润: 3480万元
- 递延收入: 超14亿元（+44%）

**盈利能力**: 规模化盈利，利润率提升

### Phase 6: 市场分歧分析

**Bull Case**:

- 净利率9.2%，盈利能力强劲
- 递延收入14亿元（+44%），未来收入保障
