### Phase 2: 行业周期分析

**行业阶段**: 疫苗行业下行周期，价格战激烈，行业调整期

**中国疫苗行业（2025年）**:
- 市场规模: 约500-600亿元
- 增速: 下滑，负增长
- 竞争格局:
  - 头部厂商: 智飞生物、万泰生物、沃森生物、康泰生物等
  - 价格战: HPV、流感等价格创历史新低
- 关键趋势:
  - 集采常态化
  - 价格战愈演愈烈
  - 新生儿出生率持续走低
  - "疫苗犹豫"现象凸显
  - 产品同质化严重
  - 海外市场成为新突破口

**HPV疫苗行业**:
- 市场规模: 约100-150亿元
- 竞争格局恶化:
  - 国产HPV疫苗崛起: 沃森生物、万泰生物、国药集团
  - 价格战: 2价HPV中标价27.5元/支（-90%）
  - 万泰9价HPV定价499元/支（仅为默沙东40%）
  - 默沙东9价HPV批签发量下降
- 政策变化: 双价HPV疫苗纳入国家免疫规划

**肺炎疫苗行业**:
- 市场规模: 约50-60亿元
- 竞争格局:
  - 沃森生物: 国内首家
  - 康泰生物: 13价肺炎疫苗上市
  - 辉瑞: 进口Prevnar 13
- 趋势: 竞争加剧，价格战

**海外市场**:
- 中国疫苗出口增长
- 性价比优势明显
- 东南亚、拉美、非洲等新兴市场

**监管环境**:
- 疫苗行业监管趋严
- 集采常态化
- 行业协会倡议"严禁低于成本价竞标"

**周期判断**: 疫苗行业进入下行周期，价格战激烈，新生儿出生率下降，企业依靠海外拓展和创新产品突围。
