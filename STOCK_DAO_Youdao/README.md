## 📋 8阶段分析摘要

### Phase 1-2: 公司与行业
**公司**: 网易有道，网易旗下智能学习公司，2019年纽交所上市
**行业**: 在线教育、智能硬件、AI教育应用
**行业阶段**: 成熟期，AI转型期
**行业趋势**: AI赋能教育、智能硬件升级、在线营销增长

### Phase 3: 业务拆解
**核心业务（2025）**:
- **学习服务 (44%)**: 26亿元，AI驱动订阅产品
- **在线营销 (42%)**: 25亿元（+28.5%），游戏广告驱动
- **智能设备 (14%)**: 7.4亿元（-18.2%），词典笔、答疑笔

### Phase 4: 财务质量
**2025年全年**:
- 营收: 59亿元（+5%）
- 经营利润: 2.213亿元（+48.7%）
- 净利润: 0.482亿元（Q4）
- 首个全年正经营现金流

### Phase 5-6: 治理与市场
**Bull Case**:
- "子曰"大模型领先
- 在线营销快速增长
- AI应用场景丰富

**Bear Case**:
- 硬件业务持续下滑
- 学习服务收入下降
- 盈利能力弱

### Phase 7: 估值与护城河
