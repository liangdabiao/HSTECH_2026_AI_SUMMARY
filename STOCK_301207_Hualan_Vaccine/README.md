## 核心投资亮点

- 流感疫苗生产规模居亚洲首位
- 全球首家研制出甲型H1N1流感疫苗
- 国内首家获批生产成人及儿童四价流感疫苗
- 中国唯一通过流感疫苗WHO认证的企业
- 2018-2020年流感疫苗批签发量占行业首位

---

## 主要风险提示

- 🔴🔴 2024年产品大幅降价（四价流感疫苗价格下调40%以上）
- 🔴🔴 2024年净利润2.06亿元，同比下降76.10%
- 🔴🔴 疫苗接种率变动，市场需求下滑
- 🔴🔴 销量减少，业绩显著下滑
