## 业务布局

### 基础教育

安博在湖南、江苏、辽宁等地拥有多所全日制中小学校，提供素质教育，培养国际竞争人才。

### 国际教育

安博国际学院提供:
- 美国学分课程
- 语言集训
- 留学申请
- 职业规划指导
- 就业推荐
- 海外管家式服务

### 高等教育

安博大学与700多家院校合作:
- 学院共建
- 专业共建
- 课程置换
- 基地共建
- 企业实训
- 师资培训
- 职业素养提升

### 企业培训

- 企业拓展训练
- 管理训练
- 企业混合式培训
- 企业大学建设

## 研究结论

安博教育是一家在美国上市的教育服务提供商，业务覆盖K12、高等教育、国际教育、企业培训等多个领域。

**核心问题**:
1. 规模相对较小，抗风险能力弱
2. 股价波动大，流动性差
3. 缺乏机构研究和覆盖
4. 面临行业竞争和政策风险

**投资建议**: 不建议普通投资者参与，仅适合高风险偏好投资者小仓位配置
