## 核心投资亮点

- 2024年营收9.62亿元（+4.21%），归母净利润2.14亿元（+5.75%）
- 2024年扣非净利润2.02亿元（+5.63%），主营盈利能力稳健
- 毛利率34.79%，净利率22.21%，盈利能力优秀
- ROE约17%，ROIC 16.96%，股东回报率优秀
- 电机业务增长17.26%，新能源汽车零部件收入占比29.54%
- ADAS主动感知清洗系统前瞻布局，获Waymo、美团等客户定点
