## 🔍 核心研究发现

### ✅ 核心优势

1. **技术领先地位**
   - 国内最早实现半绝缘型SiC衬底产业化
   - 掌握核心晶体生长技术
   - 产品质量达到国际先进水平

2. **国产替代机遇**
   - 受益于半导体自主可控政策
   - 下游客户国产化意愿强
   - 供应链安全需求推动

3. **市场空间广阔**
   - 新能源汽车SiC需求爆发式增长
   - 5G基站建设拉动半绝缘型衬底需求
   - 光伏、轨道交通等应用持续渗透

### ⚠️ 核心风险

1. **持续亏损**
   - 2022-2024年持续亏损
   - 盈利拐点尚不明确
   - 现金流压力存在

2. **竞争加剧**
   - Wolfspeed等国际巨头扩产
   - 国内多家企业进入SiC领域
   - 价格战可能开启

3. **技术迭代风险**
   - 大尺寸（8英寸）技术升级压力
   - 新材料（如Ga₂O₃）潜在替代
   - 设备投资巨大

4. **客户集中度**
   - 前五大客户收入占比较高
   - 单一客户依赖风险
   - 下游需求波动影响大

## 🎯 投资建议

### 🔴🔴🔴 观望评级（当前不建议买入）

**理由：**

1. **盈利不确定性高**
   - 公司处于亏损状态，盈利路径不清晰
   - 行业价格战可能进一步恶化盈利能力
   - 现金流状况需持续关注

2. **估值缺乏安全边际**
   - PS估值偏高（~10-15倍）
   - 无法用PE估值（亏损状态）
   - DCF模型对长期增长率假设高度敏感

3. **等待更好的买入时机**
   - 建议等待盈利拐点确认
   - 或等待市场恐慌情绪导致价格大幅回调
   - 或等待8英寸技术突破的明确信号

### 🟡 潜在买入观察点

如果出现以下情况，可考虑买入：

1. **盈利改善信号**
   - 单季度扭亏为盈
   - 毛利率持续提升至30%+
   - 经营现金流转正

2. **技术突破**
   - 8英寸SiC衬底量产成功
   - 客户认证通过并获得大单
   - 良率显著提升

3. **市场机遇**
   - 行业整合，竞争对手退出
   - 下游需求超预期爆发
   - 政策强力支持
