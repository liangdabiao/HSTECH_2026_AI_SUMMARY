## 核心投资亮点

- 2024年营收57.36亿元（+3.87%），营收保持增长
- 毛利率73.74%，保持高位，盈利空间大
- 薇诺娜品牌知名度高，敏感肌肤赛道龙头
- 线上渠道强势

## 主要风险提示

- 🔴🔴 2024年归母净利润5.03亿元（-33.53%），业绩大幅下滑
- 🔴🔴 2024年扣非净利润2.40亿元（-61.12%），主营盈利能力大幅恶化
- 🔴🔴 ROE 8.53%（-4.68pct），股东回报率下降
- 🔴🔴 "双11"等线上大促活动销售不如预期
- 🔴🔴 市场竞争加剧，营销费用增加
- 🔴🔴 悦江投资业绩承诺未能完成，计提减值
- 🔴🔴 获客成本上升
- 🔴🔴 商誉减值9069万元
