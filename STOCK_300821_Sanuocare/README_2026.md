## 业绩下滑原因分析

### 1. 商誉减值 (影响最大)

**PTS公司商誉减值**: 1.3-1.7亿元

**减值原因**:
- 全资子公司PTS经营业绩未达预期
- 行业市场变化
- 预期经营情况调整
- 谨慎性原则判断

### 2. 专利和解费用

**与罗氏和解**:
- 美国子公司THI与罗氏达成和解
- 签订交叉专利许可和解协议
- 向罗氏净支付1900万美元
- 影响净利润约7463万元

### 3. 产品结构变化

**新产品投放**:
- CGM和BGM新品投放加大
- 产品结构变化影响整体毛利率
- 新产品推广初期毛利率低

**竞争加剧**:
- 行业竞争加剧
- 市场费用投入增加
- 提升新产品认知度

### 4. 费用投入增加

**市场推广**:
- 巩固市场
- 开拓新市场
- 费用投入进一步增加
