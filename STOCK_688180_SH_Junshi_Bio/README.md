## 🎯 快速摘要（3分钟版）

### 核心投资论点

**看多理由：**
1. ✅ **国际化先锋**：首个中国研发制造的生物药获FDA批准，填补美国鼻咽癌治疗空白
2. ✅ **适应症拓展**：特瑞普利单抗已在中国获批7项适应症，覆盖多个高发癌种
3. ✅ **研发管线丰富**：涵盖肿瘤、自身免疫、代谢疾病等多个治疗领域
4. ✅ **海外合作网络**：与Coherus、Hikma、Dr. Reddy's等建立全球商业化合作

**看空理由：**
1. ⚠️ **PD-1市场内卷**：国内已有10余款PD-1/PD-L1获批，竞争极度激烈
2. ⚠️ **持续亏损**：2024年归母净利润-12.81亿元，盈利路径不明晰
3. ⚠️ **市场份额落后**：特瑞普利单抗销售9.19亿元，远低于百济神州的38亿元
4. ⚠️ **现金流压力**：研发投入虽缩减但仍达12.75亿元，现金消耗持续

### 护城河评估（0-5分）

| 维度 | 评分 | 说明 |
|------|------|------|
| **品牌与专利** | 3/5 | 特瑞普利单抗首个获FDA批准的中国生物药，但专利保护有限 |
| **监管壁垒** | 3/5 | 已获中美批准，但后续适应症审批面临竞争 |
| **研发能力** | 4/5 | 强大的研发团队和管线储备，研发投入占营收65% |
| **网络效应** | 2/5 | 医院和医生网络仍在建设中 |
| **成本优势** | 2/5 | 价格竞争激烈，成本优势不明显 |
| **综合护城河** | **2.8/5** | **中等偏弱** - 研发能力较强但面临激烈竞争 |

## 🔍 研究方法说明

本报告基于以下数据源和分析框架：

### 数据来源
1. **公司披露**：年度报告、季度报告、公告
2. **监管文件**：上交所、港交所披露
3. **行业研究**：券商研报、行业报告
4. **新闻资讯**：公司官网、财经媒体

### 分析框架
- **8阶段投资尽调流程**：覆盖业务基础、行业分析、业务拆解、财务质量、公司治理、市场情绪、估值护城河
- **交叉验证**：利润vs现金流、公司vs同行、多空论点对冲
- **情景分析**：乐观/基准/悲观三种假设下的估值区间
