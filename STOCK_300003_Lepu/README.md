### Phase 2: 行业周期分析

**行业阶段**: 医疗器械行业集采常态化，医美行业进入调整期

**中国医疗器械行业（2025年）**:
- 市场规模: 约8000-10000亿元
- 增速: 集采后恢复增长，增速5-10%
- 竞争格局:
  - 头部厂商: 迈瑞医疗、乐普医疗、威高股份等
  - 集采影响: 价格大幅下降，头部企业获益
- 关键趋势:
  - 集采常态化
  - 创新支持力度加大
  - 国产替代加速
  - 出海成为新方向

**心血管器械行业**:
- 市场规模: 约500-600亿元
- 集采影响: 冠脉支架、球囊等已纳入集采
- 竞争格局: 乐普医疗、微创医疗、吉威医疗等
- 关键趋势: 集采后价格稳定，头部企业市场份额提升

**医美行业（2025年）**:
- 市场规模: 约2000-2500亿元
- 增速: 调整期，增速放缓至5-10%
- 竞争格局:
  - 头部厂商: 爱美客、华熙生物、华东医药等
  - 童颜针: 国内已有7款获批，乐普医疗后来者
- 关键趋势:
  - 价格战愈演愈烈
  - 监管趋严
  - 合规化成为趋势
  - 消费需求理性化

**创新药行业（GLP-1）**:
- 市场规模: 约100-150亿元（中国）
- 增速: 高增长，年增速50%+
- 竞争格局: 诺和诺德、礼来领先，国内企业快速跟进
- 关键趋势: GLP-1减肥药需求爆发

**监管环境**:
- 集采常态化
- 医美监管细则持续细化
- 创新器械审批加速
- 脑机接口列为未来产业

**周期判断**: 医疗器械集采后恢复增长，医美行业调整期，乐普医疗依靠医美新业务和创新药实现业绩爆发。
