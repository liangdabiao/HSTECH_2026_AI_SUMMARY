## 核心投资亮点

- 2024年营收4.48亿元（+44.62%），营收大幅增长
- 2024年上半年扭亏为盈，净利润2100-2800万元
- 深潜式加压控藻成套设备全年销售7套，贡献营收1.81亿元
- 在手订单约3.67亿元，业务基础扎实
- 星云湖项目二审胜诉，或解除约1亿元资金返还义务
- 经营现金流8920万元，现金流改善
- 连续多年分红，累计分红1.62亿元

## 主要风险提示

- 🟡🟡 2024年归母净利润亏损8615万元，亏损扩大
- 🟡🟡 扣非净利润亏损9931万元，主营盈利能力弱
- 🟡🟡 星云湖项目诉讼导致计提预计负债4808万元
- 🟡🟡 应收账款6.9亿元，坏账准备2.6亿元，坏账比例高
- 🟡🟡 客户集中度超70%，前五大客户销售占比71.17%
- 🟡🟡 2020年以来净利润持续下滑
- 🟡🟡 2025年一季度净利润继续下降
