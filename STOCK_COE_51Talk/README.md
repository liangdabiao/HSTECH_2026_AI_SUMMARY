## 公司概况

### 基本信息

- **成立时间**: 2011年
- **上市时间**: 2016年
- **业务模式**: 1对1在线英语教育
- **目标市场**: K12学生
- **运营区域**: 中国香港、中东北非、东南亚

### 战略转型

**背景**: 中国监管政策变化("双减")
**行动**: 
- 战略性重心转向海外市场
- 完全剥离中国大陆业务
- 将成功经验复制到新市场
