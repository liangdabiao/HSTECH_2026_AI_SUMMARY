## 业绩亮点分析

### 1. 业绩超预期

**2024年下半年营收370亿元 (+13.4%)**:
- 超出市场普遍预期2.5%
- 增速高于市场预期的10.7%
- 斐乐和其他品牌强于预期
- 安踏品牌增长符合预期

### 2. 多品牌战略成功

**品牌矩阵完整**:
- **安踏**: 专业运动, 增长8.1%
- **斐乐**: 运动时尚, 增长5.4%, 超预期
- **迪桑特、可隆**: 高端专业, 高增长
- **亚玛芬**: 始祖鸟等高端户外, 大中华区+43.4%

### 3. 亚玛芬体育爆发

**2025年营收65.66亿美元 (+27%)**:
- 大中华区全年收入+43.4%
- Q4大中华区+41.8%
- 始祖鸟母公司强劲增长
- 安踏军团旗下品牌表现亮眼

### 4. 毛利率行业领先

**分品牌毛利率**:
- 安踏核心品牌: 52.6%
- 斐乐: 65.5%
- 其他品牌: 71.8%
- 综合: 60.4%

### 5. 股东回报丰厚

**派息率51.4%**:
- 不含AmerSports股权收益
- 与市场预期相符
- 稳定分红政策

### 6. 运营费用控制优秀
