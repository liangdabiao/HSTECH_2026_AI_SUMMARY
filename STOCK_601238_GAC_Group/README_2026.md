## 业绩恶化分析

### 1. 由盈转亏

**2025年预计净亏损80-90亿元**:
- 2024年净利润8.24亿元
- 一年之间亏损近100亿
- 扣非净利润连续两年亏损

### 2. 销量大幅下滑

**全年销量172.15万辆 (-14.06%)**:
- 新能源车43.36万辆 (-4.64%)
- 多个主力车型受价格战影响
- 销量未达预期

### 3. 自主品牌全面溃败

**四大品牌三跌一涨**:
- 广汽本田: -25.22% (最惨)
- 广汽传祺: -23.02%
- 广汽埃安: -22.62%
- 广汽丰田: +2.44% (唯一增长)

### 4. 新能源转型失败

**重点新能源车型未达目标**:
- 销量处于爬坡期
- 未达到计划目标
- 与华为合作尚无实质产品

### 5. 渠道落后

**销售体系结构性错配**:
- 以传统4S店为主导
- 直营、代理、互联网等新渠道建设落后
- 营销体系效能提升慢

### 6. 高管动荡

**多位高管被带走**:
- 内部调整和清算
- 利益输送链条问题
- 管理层不稳定

## 业务分析

### 1. 品牌矩阵

**合资品牌**:
- 广汽丰田: 唯一增长点, 铂智3X销量83025辆
- 广汽本田: 大幅下滑25.22%

**自主品牌**:
- 广汽传祺: 销量下滑23.02%
- 广汽埃安: 销量下滑22.62%
- 昊铂: 与埃安整合为昊铂埃安BU

### 2. 新能源转型

**当前困境**:
- 重点新能源车型销量处于爬坡期
- 未达计划目标
- 与华为合作首款车型计划2026年推出

**与华为合作**:
- 2025年3月成立华望汽车
- 首款车型定位于30万级
- 计划2026年推出

### 3. 海外市场

**亮点**:
- 自主品牌海外销量近13万辆 (+47%)
- 但基数较小, 难以扭转整体颓势
- 已进入84个国家和地区

### 4. 战略调整

**三年"番禺行动"**:
- 自主品牌BU改革
- 稳合资、强自主、拓生态
- 战时状态推进变革
