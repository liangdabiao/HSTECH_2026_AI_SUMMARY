## 公司概览

### 基本信息

| 项目 | 内容 |
|------|------|
| **公司名称** | 比亚迪股份有限公司 (BYD Co., Ltd.) |
| **股票代码** | 01211.HK / 1211.HK / 002594.SZ |
| **上市时间** | 2002年(H股)、2011年(A股) |
| **总部** | 中国深圳 |

### 业务演进

- **1995-2002**: 电池业务起家，成为全球领先电池供应商
- **2003**: 进入汽车行业，收购西安秦川汽车
- **2008**: 推出首款新能源汽车F3DM
- **2020**: 推出刀片电池，汉EV上市
- **2021**: DM-i超级混动发布，销量爆发
- **2022**: 停售燃油车，全面电动化
- **2025**: 全球新能源汽车销量第一

## 核心业务

### 1. 乘用车业务 (主要收入来源)

#### 新能源汽车 (NEV)

**品牌矩阵**:

| 品牌 | 定位 | 代表车型 | 2025年表现 |
|------|------|----------|------------|
| **比亚迪主品牌** | 大众市场 | 秦、汉、唐、宋、元、海豚、海豹 | 销量主力 |
| **腾势 (DENZA)** | 高端 | D9、N7、N8 | 2026年出海 |
| **仰望 (YANGWANG)** | 超豪华 | U8 (百万级越野) | 品牌向上 |
| **方程豹** | 专业越野 | 豹5 | 细分市场 |

**技术路线**:

| 类型 | 技术代表 | 市场地位 |
|------|----------|----------|
| **纯电(BEV)** | e平台3.0、刀片电池 | 全球领先 |
| **混动(PHEV)** | DM-i/DM-p超级混动 | 市场绝对领先 |
| **插混(EREV)** | DM-i | 成本优势显著 |

### 2. 动力电池业务

**产品矩阵**:
- **刀片电池**: 第二代已发布，全球领先安全性和能量密度
- **储能电池**: Grid Pro、Module C
- **电池回收**: 梯次利用、材料回收

**市场地位**:
- 全球动力电池装机量第二
- 2025年装机量超200GWh
- 外供比例提升(特斯拉、丰田等)

### 3. 电子业务

**产品范围**:
- 智能手机零部件(摄像头、机壳)
- 智能产品(平板、穿戴设备)
- 新型智能产品(机器人、无人机)
- 新能源汽车零部件

**主要客户**:
- 苹果、华为、小米、OPPO、vivo

### 4. 其他业务

- **半导体**: 车规级IGBT、SiC芯片
- **轨道交通**: 云轨、云巴
- **新能源**: 光伏、储能

## 价值链位置

### 垂直整合模式 (核心竞争优势)

```
上游: 锂、钴、镍等矿产资源
      ↓ (部分布局)
中游: 电池材料(正极、负极、电解液)
      ↓ (自研自产)
电芯: 刀片电池(自研自产)
      ↓ (垂直整合)
核心零部件: 电机、电控、半导体(自研自产)
      ↓ (垂直整合)
整车: 研发、生产、销售
      ↓ (直营+经销商)
下游: 充电网络(闪充站)、后市场服务
```

### 价值创造

- **垂直整合**: 成本优势、供应链安全
- **技术自研**: 刀片电池、DM-i等核心技术
- **规模效应**: 460万辆规模，成本摊薄

## 战略定位

### 公司战略

1. **技术为王**
   - 刀片电池持续迭代
   - 智能化加大投入
   - 半导体的车规级芯片

2. **品牌向上**
   - 仰望冲击百万级市场
   - 腾势站稳高端市场
   - 主品牌升级

3. **全球化布局**
   - 2025年海外销量104万辆
   - 2026年目标130万辆
   - 本地化生产(泰国、巴西、匈牙利)

4. **全产业链覆盖**
   - 上游资源布局
   - 中游核心零部件自研
   - 下游充电网络建设

### 竞争策略

| 策略 | 说明 |
|------|------|
| **成本领先** | 垂直整合，规模效应 |
| **技术差异化** | 刀片电池、DM-i |
| **品牌阶梯化** | 多品牌覆盖全市场 |
| **快节奏迭代** | 车型更新速度领先 |

## 核心竞争力

### 优势

1. **垂直整合**: 全产业链布局，成本优势显著
2. **技术领先**: 刀片电池、DM-i、e平台3.0
3. **规模优势**: 460万辆，全球第一
4. **品牌矩阵**: 多品牌覆盖全价格带
5. **快速迭代**: 车型推出速度快

### 劣势

1. **智能化**: 相对华为、小鹏仍有差距
2. **海外品牌**: 认知度仍需提升
3. **高端市场**: 仰望、腾势仍在培育期
4. **盈利压力**: 价格战持续
