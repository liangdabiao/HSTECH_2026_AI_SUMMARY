## 核心发现摘要

### 投资亮点

1. **全球新能源汽车龙头**: 2025年销量460万辆，超越特斯拉
2. **海外市场爆发**: 2025年海外销量104万辆，同比+152%
3. **技术护城河深厚**: 刀片电池、DM-i超级混动、e平台3.0
4. **垂直整合优势**: 全产业链布局，成本优势显著
5. **盈利能力强**: Q1营业利润+90.06%，现金流极佳

### 主要风险

1. **价格战持续**: 毛利率承压
2. **海外政策风险**: 欧盟反补贴、美国关税
3. **智能化落后**: 相对华为、小鹏有差距
4. **产能过剩**: 行业竞争加剧
