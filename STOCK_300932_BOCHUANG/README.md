## 核心投资亮点

- 2024年营收21.68亿元（+18.07%），净利润6332万元（+36.48%）
- 2025年前三季度营收17.45亿元（+9.03%），净利润6288万元（+22.20%）
- 新项目孵化贡献增量，2024年通过31家客户审核及光储企业导入
- 海外战略成效显现，海外业务销售占比同比增长29.73%
- 2024年经营现金流2.04亿元，现金流良好
- 分红方案：每10股派发现金红利2.18元，转增4股
