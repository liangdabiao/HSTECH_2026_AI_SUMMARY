**关键里程碑**:
- 2006年: 公司成立

### Phase 2: 行业周期分析

**行业阶段**: 光伏行业深度调整期，衰退期

**全球光伏行业（2025年）**:
- 市场规模: 约500GW装机量/年
- 增速: 放缓至5-10%（前几年30%+）
- 产能过剩: 严重（产能2x需求）
- 竞争格局:
  - 头部厂商: 晶科能源、隆基绿能、天合光能、晶澳科技、阿特斯
  - 全行业亏损
- 关键趋势:
  - 产能过剩严重
  - 价格战持续（组件价格0.6元/W）
  - 行业深度调整
  - 技术迭代加速（N型TOPCon替代P型PERC）
  - 海外贸易壁垒加剧
  - 现金流压力剧增

**中国光伏行业（2025年）**:
- 市场规模: 约200GW装机量/年
- 增速: 放缓
- 竞争格局:
  - 头部厂商: 晶科、隆基、天合、晶澳、阿特斯、通威
  - 全行业亏损
- 关键趋势:
  - 反内卷共识
  - 产能出清加速
  - 并购重组开始
  - 2026年或成分水岭

**光伏组件行业（2025年）**:
- 市场规模: 约3000亿元
- 价格: 0.6-0.8元/W（较2023年下降40%）
- 产能: 严重过剩
- 关键趋势:
  - N型TOPCon技术成为主流
  - 大尺寸组件（182mm、210mm）
  - 高功率组件（600W+）

**监管环境**:
- 海外贸易壁垒（美国、欧盟反倾销）
- 国内电力市场改革
- ESG要求提升

**周期判断**: 光伏行业深度调整期，产能过剩严重，价格战持续，全行业亏损，2026年或成分水岭。
