### Phase 2: 行业周期分析

**行业阶段**: 中国疫苗行业成长期，新冠疫苗行业衰退期

**中国疫苗行业（2025年）**:
- 市场规模: 约500-600亿元
- 增速: 中速增长，年增速10-15%
- 竞争格局:
  - 头部厂商: 智飞生物、沃森生物、康泰生物、康希诺等
  - 康希诺在创新疫苗领域领先
- 关键趋势:
  - 国产替代加速
  - 多联多价疫苗趋势
  - 成人疫苗接种意识提升
  - 出海加速

**新冠疫苗行业（2025年）**:
- 市场规模: 约50-100亿元
- 增速: 大幅下滑
- 竞争格局:
  - 头部厂商: 科兴、国药、康希诺、智飞生物
  - 竞争激烈，价格战
- 关键趋势:
  - 需求大幅下降
  - 加强针市场萎缩
  - 全球疫苗过剩
  - 技术路线分化

**监管环境**:
- 疫苗管理法
- 批签发制度
- 医保谈判
- 集采降价（部分疫苗）
- 国际注册要求

**周期判断**: 疫苗行业成长期，新冠疫苗衰退期，康希诺从新冠疫苗向常规疫苗转型。
