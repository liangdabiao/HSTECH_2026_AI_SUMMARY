## 核心投资亮点

- 2024年营收9.27亿元（+14.69%），归母净利润1.81亿元（+100.11%）
- 2024年扣非净利润1.16亿元（+54.68%），主营盈利能力改善
- 2024年上半年毛利率37.09%（+3.74pct），盈利能力提升
- 业务结构优化，产品销售和运营服务占比提升
- 与紫金矿业合作深化，关联交易预计4.5亿元
- 股权激励目标锁定2025-2027年高增长

---

## 主要风险提示

- 🟡🟡 2024年归母净利润增长主要依赖投资收益6163万元
- 🟡🟡 2025年Q1归母净利润1800万元（-77.68%），业绩大幅下滑
- 🟡🟡 与紫金矿业关联交易占比高，客户集中度风险
- 🟡🟡 收购紫金药剂形成商誉1.69亿元
- 🟡🟡 2025年上半年业绩预告扣非净利润下降11%-17%
- 🟡🟡 无业绩对赌，并购整合风险
