## 💡 核心发现

### ✅ 优势 (5个关键优势)

1. **AI算力需求爆发**
   - 2025年新订单65%与AI相关
   - 阿里巴巴3,800亿元AI投资
   - 瑞银预测:阿里云2032年耗电量增加10倍

2. **盈利拐点已至**
   - 毛利率: 19.9% → 21.5% (+1.6pp)
   - 亏损: 39.26亿 → 7.7亿 (-80%)
   - 2025年预计扭亏为盈

3. **宽阔护城河**
   - 转换成本高(长期合同+数据迁移成本)
   - 成本优势(规模效应,EBITDA率47.2%行业领先)
   - 资源优势(核心城市土地+电力稀缺)

4. **历史低估值**
   - EV/EBITDA 10x(历史低位)
   - 较历史平均低30-35%
   - 投行目标价45-50港元,上涨空间45-55%

5. **自由现金流转正**
   - 2024年FCF约-3亿元(大幅改善)
   - 2025年预计转正(+2亿元)
   - 2026年稳定为正(+10亿元)

### ⚠️ 风险 (5个关键风险)

1. **高额资本开支**
   - 2024-2025年Capex约43亿元/年
   - 压制自由现金流
   - AI高密度机柜需要更高Capex

2. **供给过剩**
   - 行业上架率<60%(万国数据73.8%)
   - 价格战持续
   - 2025年后AI需求消化供给

3. **大客户依赖**
   - 前10大客户占比>50%
   - 议价能力强,毛利率受限
   - 客户流失风险高

4. **中概股风险**
   - 地缘政治不确定性
   - 监管风险(数据安全、审计底稿)
   - 估值折价30-50%

5. **AI需求不及预期**
   - 管理层预警2026年新增订单走弱
   - 云厂商AI资本开支可能削减
   - 国产AI芯片进度落后
