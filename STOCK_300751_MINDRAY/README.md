## 核心投资逻辑

### 1. 行业地位绝对领先

根据弗若斯特沙利文数据:
- 中国医疗器械市值规模超过第二到五名之和
- 工业收入是第二名的两倍有余
- 全球顶尖医疗器械企业第23名
- 全球前30大器械企业中唯一上榜的中国企业

### 2. 产品矩阵完整

- 体外诊断 (IVD): 中国最大、全球第二大血球诊断提供商
- 生命信息与支持: 中国最大、全球第二大监护仪提供商
- 医学影像: 全球第三、中国第一超声影像供货商
- 微创外科/介入: 新兴业务快速增长

### 3. 三大战略方向

- **流水型业务**: 试剂、高值耗材、数字化服务
- **国际化**: 国际业务占比超50%，持续提升
- **数智化**: 设备+IT+AI，发布多款医学大模型

### 4. 2030年战略目标

成为全球医疗器械综合实力Top 10
