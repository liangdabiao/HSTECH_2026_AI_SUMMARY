## 🎯 核心投资论点

### 优势

1. **市场地位领先** - 全球智能电视/机顶盒SoC芯片市场领先者，中国OTT芯片市场份额第一
2. **产品线完整** - S系列(机顶盒)、T系列(TV)、C系列(AI摄像头)五大产品线布局完整
3. **客户资源优质** - 服务TCL、海信、创维、小米等主流电视厂商
4. **技术积累深厚** - 专注多媒体SoC芯片15年+，视频编解码技术领先
5. **新业务拓展** - AIoT、汽车电子芯片打开第二增长曲线

### 风险

1. **市场竞争加剧** - 联发科(Mstar)、瑞昱等竞争对手实力强大
2. **客户集中度较高** - 前五大客户占比较高，客户流失风险
3. **研发投入压力** - 需持续高研发投入保持技术领先
4. **行业周期性** - 消费电子需求波动影响业绩
5. **供应链依赖** - Fabless模式依赖晶圆代工厂产能

### 投资建议

**适合投资者**: 成长型投资者，可承受中高风险，持有周期1-3年

**推荐买入区间**:
- 保守投资者: 等待回调至PE 25x以下
- 积极投资者: 当前价位可分批建仓

**关键监控指标**:
- 智能电视/机顶盒出货量增速
- AIoT新业务收入占比
- 毛利率变化趋势
- 研发投入占比
- 主要客户合作稳定性
