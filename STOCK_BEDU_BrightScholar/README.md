## 研究结论

### 退市原因分析

1. **估值低迷**: 股价长期低于1美元，市值仅6,675万美元
2. **私有化溢价**: 收购价0.575美元，较IPO价格10.5美元下跌94.5%
3. **碧桂园困境**: 母公司碧桂园财务危机影响
4. **行业转型**: 教育"双减"政策后业务转型困难

### 历史财务表现 (退市前)

| 指标 | 2025财年Q2 | 2024财年 | 2023财年 |
|------|-----------|---------|---------|
| 营收 | 5,520万美元 | 下降趋势 | 19-20亿元 |
| 净利润 | 403万美元 | 波动较大 | 亏损为主 |
| 毛利率 | 30.0% | 32.9%-36.6% | 下降趋势 |
| 在校生 | 2,669人 | - | 5.8万人 |

### 业务结构 (退市前)

1. **海外学校** (47.8%): 8所英国/美国学校，增长引擎
2. **留学咨询** (27.2%): 增长亮点，同比增长6.2%
3. **国内K12** (21.6%): 受政策影响，收入下降23.9%

## 投资者启示

### 成功案例
- 2017年IPO: 10.5美元/股，募资1.575亿美元
- 海外扩张: 成功收购10所海外学校
- 业务转型: 从K12转向国际教育和留学服务

### 失败教训
- 政策风险: "双减"导致剥离68所幼儿园和23所学校
- 估值崩溃: 从IPO 10.5美元跌至退市0.575美元
- 母公司拖累: 碧桂园危机传导
- 现金流压力: 经营现金流持续为负
