## 1. 医药自营业务

### 业务模式
**直接面向消费者销售药品和健康产品**，通过"阿里健康大药房"和"阿里健康海外旗舰店"运营。

### 产品结构
| 产品类别 | 占比 | 特点 | 增长情况 |
|----------|------|------|----------|
| 处方药 | ~35% | 政策限制，增长潜力大 | 高增长 |
| OTC药品 | ~30% | 标准化产品，价格竞争激烈 | 稳定增长 |
| 医疗器械 | ~15% | 高客单价，专业性强 | 中速增长 |
| 保健品 | ~12% | 消费属性强，毛利高 | 稳定增长 |
| 其他健康产品 | ~8% | 个护、母婴等 | 稳定增长 |

### 核心数据
- **年度活跃消费者**: 6500万+
- **SKU数量**: 数万种药品和健康产品
- **配送时效**: 24-48小时(标准)，30分钟(部分城市即时达)
- **复购率**: 慢性病患者复购率超60%

### 竞争优势
1. **阿里生态流量**: 淘宝、天猫流量导入
2. **供应链管理**: 直采模式，质量可控
3. **价格优势**: 规模采购，成本优势
4. **品牌信任**: 阿里品牌背书

### 盈利能力
- **毛利率**: ~25%
- **运营成本**: 物流、仓储、人力
- **净利润率**: ~5-6%

### 增长驱动因素
1. ✅ **处方药网售放开**: 政策逐步放开
2. ✅ **慢性病管理**: 慢病患者需求增长
3. ✅ **用户习惯养成**: 疫情加速线上购药习惯
4. ✅ **价格优势**: 相比线下药店价格更低

### 风险挑战
1. ❌ **价格竞争**: 与京东健康等价格战持续
2. ❌ **政策风险**: 处方药网售政策不确定性
3. ❌ **毛利率压力**: 价格战挤压毛利率
4. ❌ **供应链成本**: 药品采购成本上升

## 2. 医药电商平台业务

### 业务模式
**第三方医药电商平台**，为其他医药商家提供线上销售渠道，收取佣金和广告费。

### 平台数据
- **服务商家**: 1.8万+
- **平台GMV**: 554亿元+
- **年活跃消费者**: 2.5亿+
- **佣金率**: ~5-8%

### 收入结构
| 收入类型 | 占比 | 说明 |
|----------|------|------|
| 交易佣金 | ~60% | 按GMV收取佣金 |
| 广告收入 | ~30% | 营销服务、推广 |
| 增值服务 | ~10% | 物流、技术等 |

### 商家结构
- **连锁药店**: 老百姓、大参林等
- **品牌药企**: 直接开设旗舰店
- **单体药店**: 区域性小药店
- **医疗器械商**: 专业医疗器械供应商

### 竞争优势
1. **平台规模**: 商家和用户规模最大
2. **流量优势**: 依托天猫流量
3. **数据能力**: 用户行为数据分析
4. **基础设施**: 支付、物流等基础设施完善

### 盈利能力
- **毛利率**: ~70% (轻资产模式)
- **运营成本**: 平台运营、技术、人力
- **净利润率**: ~15-20%

### 增长驱动因素
1. ✅ **商家线上化**: 传统药店线上化需求
2. ✅ **品类扩张**: 医疗器械、保健品等品类扩张
3. ✅ **服务升级**: 营销、数据等增值服务
4. ✅ **下沉市场**: 三四线城市商家增加

### 风险挑战
1. ❌ **商家流失**: 京东健康等平台竞争
2. ❌ **佣金压力**: 商家佣金承受能力有限
3. ❌ **质量管控**: 第三方商家质量管控难度大
4. ❌ **政策风险**: 平台责任加大

## 3. 医疗健康及数字化服务

### 业务板块构成

#### 3.1 互联网医院
- **牌照**: 已获得互联网医院牌照
- **服务**: 在线问诊、电子处方、远程医疗
- **医生资源**: 数万签约医生
- **用户**: 年使用量千万级

#### 3.2 在线问诊
- **服务模式**: 图文、电话、视频问诊
- **收费**: 免费+付费混合
- **医生**: 三甲医院医生为主
- **用户**: 年轻群体、慢性病患者

#### 3.3 消费医疗
- **服务内容**: 体检、医美、口腔等
- **商业模式**: 导流+分成
- **合作伙伴**: 美年健康、爱尔眼科等
- **增长**: 年增速30%+

#### 3.4 AI医疗
- **核心产品**: 医学大模型"氢离子"
- **定位**: 医生版GPT，辅助诊疗
- **战略**: 三年不考虑商业化
- **应用**: 临床决策支持、科研辅助

### 核心数据
- **收入**: 12亿元+ (2025财年中期)
- **增速**: 30%+
- **毛利率**: ~40%
- **用户渗透**: 占总用户比例低，潜力大

### 竞争优势
1. **AI技术**: "氢离子"大模型领先
2. **阿里协同**: 阿里云技术支持
3. **数据积累**: 用户健康数据积累
4. **医生资源**: 签约医生规模大

### 盈利能力
- **毛利率**: ~40%
- **运营成本**: 医生成本、技术投入、运营
- **净利润率**: 亏损阶段

### 增长驱动因素
1. ✅ **AI技术突破**: "氢离子"商业化潜力
2. ✅ **慢病管理**: 慢病患者需求增长
3. ✅ **消费升级**: 健康管理需求增长
4. ✅ **政策支持**: 互联网医疗政策支持

### 风险挑战
1. ❌ **盈利难题**: 短期难以盈利
2. ❌ **技术风险**: AI医疗应用落地不确定性
3. ❌ **医生资源**: 医生资源获取成本高
4. ❌ **用户习惯**: 用户付费意愿不强

## 业务协同效应

### 三大业务协同
```
医药自营 ←→ 医药电商平台
    ↓            ↓
    医疗健康及数字化服务
```

### 协同效应
1. **流量协同**: 自营业务用户向平台转化
2. **数据协同**: 用户行为数据共享
3. **服务协同**: 购药+医疗服务闭环
4. **品牌协同**: 阿里健康品牌统一

### 阿里生态协同
1. **流量**: 淘宝、天猫流量导入
2. **技术**: 阿里云AI、大数据支持
3. **物流**: 菜鸟网络物流支持
4. **支付**: 支付宝支付便利

## 总结

阿里健康的三大业务板块形成了"医药自营+平台服务+医疗服务"的业务格局，各有特色，相互协同。

**核心优势**: 业务协同效应强，阿里生态支持全面
**核心挑战**: 医疗服务盈利难，竞争持续激烈

**投资亮点**: 医药电商稳健增长，AI医疗未来可期
