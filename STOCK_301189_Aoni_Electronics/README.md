## 核心投资提示

⚠️ **重大风险警告：**

1. **连续两年亏损**：2023年亏损2,257.66万元，2024年预计亏损7,000-8,800万元
2. **业绩恶化趋势**：亏损幅度扩大3-4倍
3. **盈利能力弱**：低毛利产品占比上升，整体毛利率下降
4. **成本压力**：产能搬迁导致折旧摊销、管理费用增加
