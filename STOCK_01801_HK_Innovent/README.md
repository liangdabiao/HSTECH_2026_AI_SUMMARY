  - 头部厂商: 信达、恒瑞、百济、君实
  - 信达达伯舒领先
- 关键趋势:
  - 医保谈判降价
  - 适应症扩展
  - 联合治疗方案
  - 出海授权

**慢病药物市场（2025年）**:
- 市场规模: 约500-800亿元
- 增速: 高速增长，年增速25-40%
- 竞争格局:
  - 头部厂商: 信达、恒瑞、诺华诺和
  - 信达GLP-1领先
- 关键趋势:
  - GLP-1爆发
  - PCSK9兴起
  - IL-23拓展
  - 医保覆盖扩大

**监管环境**:
- 医保谈判常态化
- 集采降价
- 审评审批加速
- 出海政策支持
- 创新药鼓励

**周期判断**: 创新药行业成长期，PD-1成熟期，慢病药物爆发期，信达生物双轮驱动战略成功。
