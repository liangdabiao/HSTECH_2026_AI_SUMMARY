  - 短视频（抖音、快手）抢占用户时长
  - 游戏厂商直接运营直播渠道

- 增速: 低增长，成熟市场
- 竞争格局:
  - 头部厂商: Twitch、YouTube Gaming、斗鱼、虎牙
  - Twitch主导全球市场

  - 移动端直播增长

**语音社交行业**:
- 市场规模: 约50-80亿元

**监管环境**:
