## 核心发现摘要

### 护城河评级: 4/5 (很强护城河)

**评级细分**:
- 规模优势: 5/5 (极强)
- 成本优势: 4/5 (强)
- 客户关系: 4/5 (强)
- 资源禀赋: 5/5 (极强)
- 监管壁垒: 5/5 (极高)

### 核心优势

1. **规模优势**: 全球最大油轮船东，运力2561万载重吨
2. **船型齐全**: 覆盖全球主流油轮船型
3. **LNG引领**: 中国LNG运输业务引领者
4. **央企背景**: 中远海运集团支持

### 主要风险

1. **行业周期性**: 油运强周期，运价波动大
2. **周期见顶**: 2026年可能进入衰退期
3. **运力过剩**: 新船交付压力
4. **地缘政治**: OPEC+、制裁政策变化

### 投资亮点

1. **油运景气上行**: 连续上升四年，9月VLCC运价暴涨
2. **LNG快速增长**: 2025H1收入+29.8%
3. **分红计划吸引**: 2025-2027年分红≥30%，成熟期可达80%
4. **券商看好**: 国泰君安/海通目标价20.26港元
