## 3.2 WPS办公软件业务

### 3.2.1 个人业务 (C端)

**产品矩阵**:
- WPS Office免费版
- WPS超级会员 (¥99/年)
- WPS AI会员 (¥199/年)
- 稻壳儿素材库

**关键指标** (2024年):

| 指标 | 数据 | 同比增长 |
|------|------|----------|
| 营收 | 32.83亿元 | +18% |
| 全球月活设备数 | 6.32亿 | - |
| 国内PC日活设备数 | 1亿+ | - |
| 国内年度付费用户 | 4170万 | +17% |
| 付费转化率 | 6.6% | - |
| ARPU | ~79元/年 | - |

**增长驱动**:
1. **用户规模增长**: 全球月活6.32亿
2. **付费转化率提升**: 从~5%到6.6%
3. **AI功能溢价**: WPS AI会员定价更高
4. **会员权益丰富**: 素材库、PDF工具、云空间

**盈利能力**:
- **毛利率**: ~85% (软件行业典型)
- **销售费用率**: ~15% (市场推广)
- **研发费用率**: ~33% (AI研发)

---

### 3.2.2 企业业务 (B端)

**产品矩阵**:
- WPS 365基础版
- WPS 365企业版
- WPS 365政务版
- 私有化部署方案

**关键数据** (2024年):
- **营收增速**: >30% (公司未披露绝对值)
- **企业客户**: 超百万家政企组织
- **央国企客户**: 覆盖率领先
- **ARPU**: ~数千元/年/企业

**核心客户**:
- 党政机关 (信创采购)
- 央企 (国企数字化)
- 金融机构 (银行、保险)
- 教育机构 (高校、K12)

**竞争优势**:
1. ✅ **信创认证**: 全栈国产化适配
2. ✅ **数据安全**: 本地化部署、国密加密
3. ✅ **价格优势**: 比Microsoft便宜50%+
4. ✅ **服务响应**: 国内本土化服务

**商业模式**:
- 订阅制 (年付/多年付)
- 按人数定价
- 定制化开发 (额外收费)

---

### 3.2.3 WPS软件授权 (一次性)

**业务模式**: 传统软件产品授权
**趋势**: **下降** (客户转向订阅制)
**原因**:
- 云端协作需求增长
- 订阅制长期成本更低
- 功能更新持续

**未来展望**: 继续下降，占比<10%

---

### 3.2.4 WPS AI商业化

**产品矩阵**:
- WPS AI写作助手
- WPS AI阅读助手
- WPS AI数据助手
- WPS AI设计助手

**商业化进展**:
- **2024年**: AI收入流水过1亿元
- **2025年**: 加速放量 (目标未披露)
- **定价**: WPS AI会员¥199/年 (比普通会员贵2倍)

**用户反馈**:
- ✅ 文档生成功能受欢迎
- ✅ 表格分析效率提升
- ⚠️ AI功能使用频率有限
- ⚠️ 付费意愿有待培养

**对标**:
- Microsoft 365 Copilot: 30美元/月 (~2400元/年)
- WPS AI: 199元/年
- **价格优势明显**

### 3.3.2 《剑网3》 - 旗舰IP

**基本信息**:
- 上线时间: 2009年 (运营15年+)
- 游戏类型: 武侠MMORPG
- 收费模式: 免费游玩+道具付费

**关键数据** (2024年):
- **营收**: 核心收入来源 (具体数据未披露)
- **增长**: 同比强劲增长
- **DAU**: 同比增长 (具体数据未披露)

**生命周期曲线**:
```
2009: 上线
2010-2013: 成长期
2014-2019: 成熟期
2020-2023: 二次增长 (云游戏、新资料片)
2024-: 长青期 (持续运营)
```

**变现模式**:
1. **外观付费**: 服装、坐骑、宠物
2. **增值服务**: VIP会员、特权
3. **资料片**: 新内容付费
4. **云游戏**: 云端订阅

**用户特征**:
- 核心用户: 25-35岁，高ARPU
- 女性用户占比: 高 (~40%)
- 用户粘性: 极高 (游戏生命周期长)

**竞争优势**:
1. ✅ **IP价值**: 15年运营沉淀
2. ✅ **社区文化**: 同人创作活跃
3. ✅ **美术风格**: 唯美古风
4. ✅ **用户粘性**: 社交关系强

## 3.4 金山云 (联营公司)

### 3.4.1 业务概述

**成立时间**: 2012年
**上市**: 2020年纳斯达克，2022年港股双重上市
**定位**: 独立云服务商，小米金山生态核心云平台

**股权结构**:
- 金山软件: 37.4%
- 小米集团: 12.25%
- 公众股东: ~50%

---

### 3.4.2 业务构成

**公有云服务** (占65%):
- 计算服务 (ECS、GPU)
- 存储服务 (OSS、NAS)
- 内容分发 (CDN)
- 数据库 (RDS、MongoDB)

**行业云服务** (占35%):
- 金融云
- 医疗云
- 政务云
- 视频云

---

### 3.4.3 财务数据 (2025Q3)

| 指标 | 金额 | 同比增长 |
|------|------|----------|
| 营收 | 24.78亿元 | +31.4% |
| 公有云收入 | 17.52亿元 | +49.1% |
| AI收入 | 7.82亿元 | 占比31% |
| 毛利率 | 15.4% | -0.7ppt |
| 经营亏损 | 1.45亿元 | 大幅收窄 |
| 净亏损 | 0.78亿元 | 大幅收窄 |
| 现金及等价物 | 40.5亿元 | - |

**关键趋势**:
1. ✅ **AI收入爆发**: 从23Q4的0.84亿到24Q3的3.6亿
2. ✅ **小米生态协同**: 收入6.9亿元 (+84% YoY)
3. ✅ **亏损收窄**: 经调整EBITDA转正
4. ⚠️ **毛利率压力**: AI算力成本高

---

### 3.4.4 战略意义

**对金山软件**:
1. **生态协同**: WPS+云+游戏
2. **AI算力支持**: WPS AI训练推理
3. **小米智能汽车**: 云服务需求
4. **未来增长点**: AI云服务

**对小米**:
1. **算力支持**: 智能车、手机、AIoT
2. **数据存储**: 用户数据管理
3. **AI推理**: 自动驾驶算法

---

### 3.4.5 盈利路径

**现状**: 持续亏损但收窄
**目标**: 2025-2026年盈亏平衡

**关键举措**:
1. **提升毛利率**: AI业务占比提升
2. **控制成本**: 缩减低毛利CDN业务
3. **生态协同**: 小米订单保证收入
4. **规模效应**: 收入增长摊薄固定成本
