## 核心发现摘要

### ✅ 投资亮点

1. **稀缺性标的**：中国大陆唯一掌握14nm及以下制程的晶圆代工企业
2. **国产替代受益者**：产业链本土化切换加速，中国区收入占比达87.6%
3. **技术突破加速**：7nm良率90%+，5nm实现技术突破
4. **产能利用率高位**：2025年平均产能利用率93.5%，Q4达95.7%
5. **业绩创新高**：2025年营收673.23亿元，净利润50.41亿元，均创历史新高

### ⚠️ 主要风险

1. **地缘政治风险**：美国制裁限制先进设备采购，EUV光刻机禁运
2. **技术差距**：与台积电、三星先进制程落后2-3代，短期难以弥合
3. **折旧压力**：2026-2027年折旧大幅增加，对毛利率形成压力
4. **估值偏高**：A股动态市盈率接近200倍，显著高于台积电（35倍）
5. **盈利波动**：2022-2024年毛利率从38.3%降至18.59%，波动较大

### 🎯 投资信号评级

**综合评级：🟡🟡（一个半绿灯）- 持有/观望**

**评级理由**：
- 生意质量：🟢 良好（技术实力强、产能满载、国产替代受益）
- 财务质量：🟡 一般（增长强劲但毛利率低、折旧压力大）
- 估值水平：🔴 偏高（A股200倍PE显著高于同行）

- A股（688981.SH）：70-80元（合理估值）

## 技术路线图

### 先进制程进展

- ✅ **14nm FinFET**（2019年量产，良率95%+）
- ✅ **N+1工艺（等效7nm）**（2023年试产，良率90%+）
- ✅ **N+2工艺（等效5nm）**（2025年突破，初期良率33%）
- 🔄 **5nm量产**（预计2026-2027年）
- 🔮 **更先进制程**（依赖EUV光刻机解禁）

### 产能扩张计划

- **2025年**：新增约5万片12英寸产能
- **2026年**：计划新增约4万片12英寸产能
- **2027年及以后**：持续扩产，但增速可能放缓
