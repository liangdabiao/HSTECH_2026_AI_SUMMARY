## 业绩亮点分析

### 1. 收入创新高

**Q3总收入16.9亿元 (+10%)**:
- 创近9个季度以来新高
- 增速提升至约10%
- 连续增长

### 2. 游戏业务爆发

**游戏相关服务5.3亿元 (+30%)**:
- 占总收入比例31.5% (首次超30%)
- 道具售卖收入+200%
- 游戏发行准备推出《鹅鸭杀》手游

**驱动因素**:
- 与游戏厂商合作加深
- 道具SKU丰富
- 《王者荣耀》《和平精英》《英雄联盟》等表现亮眼

### 3. 直播业务回暖

**直播收入11.6亿元 (+3%)**:
- 自2021年Q3以来同比增速首次转正
- 户外直播时长与收入良好增长
- 回流头部主播数量明显增长

### 4. 盈利改善

**Non-GAAP净利润3630万元**:
- 盈利能力改善
- Non-GAAP运营利润630万元

### 5. 用户规模

**总MAU 1.62亿**:
- 保持稳定
- 平台生态韧性
- 站外可触达用户破1亿
