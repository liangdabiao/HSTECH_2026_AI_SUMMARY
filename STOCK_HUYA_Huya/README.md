## 业务分析

### 1. 直播业务 (核心基本盘)

**Q3表现**:
- 收入: 11.6亿元
- 状态: 企稳回升
- 地位: 基本盘稳定

**特点**:
- 聚焦游戏直播
- 头部主播回流明显
- 品牌影响力巩固

### 2. 游戏相关服务 (第二增长曲线)

**Q3表现**:
- 收入: 5.3亿元
- 同比增长: 约30%
- 占总收入: 31.5%(首次超30%)

**驱动因素**:
- 道具售卖表现亮眼
- 《王者荣耀》《和平精英》《英雄联盟》等游戏道具销售
- 深化与游戏厂商合作
- 丰富道具SKU

**战略意义**:
- 标志着战略转型关键突破
- 收入结构多元化
- 新增长点确立

### 3. 发行业务

**重磅动作**:
- 首款发行游戏:《鹅鸭杀》手游
- 测试阶段预约量破1000万
- 体系化验证首次成功
- 为后续发行业务奠定基础
