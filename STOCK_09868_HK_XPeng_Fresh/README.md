## 🎯 核心投资逻辑

### 投资亮点(做多理由)

**1. 智能驾驶技术领先** ⭐⭐⭐⭐⭐
- 全球首个端到端大模型量产上车
- XNGP覆盖全国283城,城市NOA渗透率84%
- 自研图灵芯片700TOPS,2025年Q2量产
- 万卡算力集群(10EFLOPS),中国车企最大

**2. 技术变现突破** ⭐⭐⭐⭐
- 与大众汽车合作,标志技术能力获国际认可
- 2024年技术服务收入50.4亿(+89%)
- 技术输出模式可复制到其他国际车企

**3. 财务拐点已现** ⭐⭐⭐⭐
- 毛利率连续6季度改善: 1.5% → 14.3%
- 净亏损收窄44%: 103.8亿 → 57.9亿
- 2024年下半年自由现金流超40亿(转正)
- 2025年Q4有望实现季度盈利

**4. 销量拐点明确** ⭐⭐⭐⭐
- 2024年交付19万辆(+34.2%)
- 2025年目标38万辆(+100%)
- MONA M03连续12个月交付破万,累计18万辆
- 产品矩阵完善: X9/MONA/P7+/G6/G7

**5. 生态布局前瞻** ⭐⭐⭐
- AI汽车+飞行汽车+人形机器人三大战略
- 小鹏汇天飞行汽车获4000订单,2026年Q2交付
- 长期成长空间大

---

### 投资风险(做空理由)

**1. 持续亏损** ⚠️⚠️⚠️
- 2024年净亏损57.9亿元
- 累计亏损超250亿元
- 2025年Q4盈利存疑(概率60%)

**2. 竞争加剧** ⚠️⚠️⚠️
- 华为ADS 3.0、理想端到端快速追赶
- 特斯拉FSD可能入华,技术碾压风险
- 纯视觉vs融合路线存在不确定性

**3. 价格战压力** ⚠️⚠️
- 新能源汽车价格战持续
- 毛利率提升空间受限
- 低价车型MONA占比提升,拉低整体毛利率

**4. 规模劣势** ⚠️⚠️
- 销量19万 vs 理想50万 vs 比亚迪300万
- 规模效应不足,成本劣势
- 盈利时间表可能推迟
