## 核心投资亮点

- 2024年业绩增长：营收8.31亿元（+20.19%），净利润7047万元（+31.37%）
- 中硼硅玻璃输液瓶登记号转"A"，可正式批量化上市
- 三期高端药包材项目是山东省重大实施类项目
- 毛利率、净利率双提升
