### 主要风险提示

⚠️ **高风险因素**：
1. **持续亏损风险** - 公司成立至今尚未盈利，何时扭亏不确定性大
2. **技术迭代风险** - Wi-Fi 7等新技术迭代需要巨额研发投入
3. **竞争加剧风险** - 面临Espressif、Nordic、Broadcom等国际巨头竞争
4. **客户集中度风险** - 大客户占比高，丢单风险大
5. **供应链风险** - Fabless模式依赖晶圆代工，供应链脆弱

## 📊 使用建议

### 适合投资者类型

✅ **适合**：
- 成长型投资者（可承受高波动）
- 长期价值投资者（相信国产替代逻辑）
- 科技行业专业投资者（理解半导体行业）

❌ **不适合**：
- 保守型投资者（要求稳定分红）
- 短期投机者（公司仍处亏损期）
- 对科技行业不了解的投资者

### 建议操作策略

**当前持有者**：
- 继续持有，但设置止损线（-20%）
- 密切关注季度盈利改善信号
- 关注新产品突破和大客户进展

**潜在投资者**：
- 等待更好的入场时机（建议回调20-30%）
- 或等待公司明确扭亏信号后再入场
- 分批建仓，控制仓位比例（建议不超过5-10%）
