## 核心投资亮点

- 2024年营收23.57亿元（+22.55%），净利润9613万元（+20.02%）
- 2025年一季度营收7.14亿元（+42.87%），净利润4301万元（+33.52%）
- 2025年前三季度营收21.64亿元（+29.3%），净利润1.02亿元（+32.58%）
- 电商为强势渠道，线上业务快速增长
- 线下渠道拓展到10省2市
- 分红方案：每10股派发现金红利1.4元
