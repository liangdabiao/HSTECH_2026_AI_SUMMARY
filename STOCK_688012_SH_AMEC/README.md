## 📊 投资亮点

### ✅ 核心优势

1. **技术领先地位**
   - CCP刻蚀设备全球领先 (7nm/5nm制程)
   - ICP刻蚀设备国内领先
   - 13年持续高研发投入

2. **国产化核心受益者**
   - 中国刻蚀设备第1名
   - 全球市占率约5% (第4-5名)
   - 下游需求旺盛

3. **增长强劲**
   - 2024年营收+44.7%
   - 刻蚀设备+54.7%
   - 过去13年CAGR超35%

4. **管理层优秀**
   - 董事长尹志尧 (行业领军人物)
   - 研发团队实力雄厚
   - 资本配置合理

### ⚠️ 主要风险

1. **估值风险**
   - 当前估值偏高
   - 需要高增长消化估值
   - 市场波动敏感

2. **地缘政治风险**
   - 美国制裁潜在影响
   - 供应链安全挑战
   - 技术出口限制

3. **客户集中度高**
   - 前五大客户占比高
   - 下游需求波动风险
   - 单一客户依赖

4. **技术迭代压力**
   - 需持续高研发投入
   - 国际竞争激烈
   - 技术路线变化风险
