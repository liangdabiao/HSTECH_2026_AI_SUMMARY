## 🎯 投资论点

### 看好理由（Bull Case）

1. **国产替代加速**: 中国基因测序市场快速增长，国产替代政策支持
2. **技术领先**: DNBSEQ技术独特，专利护城河强大（700+专利）
3. **全产业链**: 唯一提供设备+试剂+软件完整解决方案的企业
4. **成本优势**: 相较Illumina价格更低，性价比优势明显
5. **专利诉讼胜诉**: 2022年获Illumina赔偿3.25亿美元，证明技术实力

### 风险因素（Bear Case）

1. **专利诉讼风险**: 与Illumina的全球专利战未完全结束
2. **地缘政治**: 美国市场受限制，海外扩张困难
3. **竞争加剧**: Illumina、Thermo Fisher等巨头持续竞争
4. **估值风险**: 科创板估值偏高，安全边际有限
5. **关联交易**: 与BGI集团关联度较高，独立性存疑

## 📈 投资建议

### 目标价位

- **乐观情景**: 基于国产替代加速和海外突破
- **基准情景**: 基于中国市场稳健增长
- **悲观情景**: 基于专利诉讼失败和海外扩张受限

### 买入/卖出时机

**建议买入**:
- 市场恐慌性回调时（-20%+）
- 重大技术突破/专利胜诉
- 海外市场重大突破

**建议卖出**:
- 估值泡沫化（P/E > 100x）
- 核心专利败诉
- 关键技术人才流失
