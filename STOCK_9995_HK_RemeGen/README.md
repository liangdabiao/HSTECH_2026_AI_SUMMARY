## 📋 8阶段分析完整版

### Phase 1: 公司事实底座

**基本信息**:
- 成立时间: 2008年
- 上市: 2020年港交所（9995），2022年科创板（688331）
- 创始人: 房健民、王威东
- 员工: 约2000人
- 总部: 中国山东烟台

**核心业务**:
1. **ADC药物**（核心业务）:
   - 维迪西妥单抗(RC48): 胃癌、尿路上皮癌
   - RC88: 卵巢癌
   - RC108: 非小细胞肺癌
2. **自身免疫药物**（核心业务）:
   - 泰它西普(RC18): 系统性红斑狼疮、类风湿关节炎
   - RC28: 糖尿病黄斑水肿
3. **眼科药物**:
   - RC28: VEGF/FGF双靶点
4. **其他生物药**:
   - 多款单抗、融合蛋白在研

**用户/客户**:
- 患者: 胃癌、尿路上皮癌、系统性红斑狼疮患者
- 医院: 全国三甲医院
- 医生: 肿瘤科、风湿免疫科医生
- 合作伙伴: Seagen(Pfizer)、Eli Lilly等

**行业地位**:
- 中国ADC药物领先企业
- 自身免疫双靶点药物创新者
- 国产创新药标杆企业

**关键里程碑**:
- 2008年: 公司成立
- 2020年: 港交所上市
- 2021年: RC48获批上市
- 2021年: RC18获批上市
- 2022年: 科创板上市

  - ADC技术突破
