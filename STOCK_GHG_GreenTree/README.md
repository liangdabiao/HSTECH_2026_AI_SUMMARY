## 业务分析

### 1. 酒店管理业务

**品牌矩阵**:
- 高端: 雅阁、雅阁璞邸(主题特色)
- 中高端: 澳斯特、格林东方、格菲系列
- 中端: 格林豪泰、格林豪泰智选
- 经济型: 都市系列、格盟、青皮树、贝壳

**规模数据** (截至2021年6月):
- 在营和筹建酒店: 近6,000家
- 覆盖城市: 中国近360座
- 个人会员: 近7,000万
- 企业会员: 近180万

**海外布局**: 美国、日本、韩国、东南亚等国家和地区

### 2. 加盟模式为主

**经营特点**:
- 加盟门店占比: 99%
- 直营门店: 30家(截至2019Q2)
- 轻资产模式
- 低经营杠杆

### 3. 中高端转型失败

**2021年策略**:
- 新开722家酒店: 189家中高端，415家中档，108家经济型
- 总负债同比+42.23%至26.24亿元

**2022年结果**:
- 营收同比下滑21.7%
- 净利润亏损3.764亿元
- 总负债进一步增至36.08亿元

## 主要风险

### 1. 业绩持续恶化

- 营收连续下滑
- 净利润大幅下降
- 运营数据持续恶化

### 2. 高负债风险

- 2022年资产负债率70.71%
- 超出60%安全范围
- 偿债压力大

### 3. 竞争劣势

- 品牌力弱于三大酒店集团
- 中高端转型失败
- 市场份额流失

### 4. 行业周期性

- 酒店业受宏观经济影响大
- 疫情后复苏不均衡
- 消费降级影响
