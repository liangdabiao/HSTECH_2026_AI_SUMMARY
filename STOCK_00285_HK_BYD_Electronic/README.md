## 💡 核心投资论点

### 四大买入理由

1. **汽车电子业务黄金期** ⭐⭐⭐⭐⭐
   - 受益母公司比亚迪销量增长(2025目标550万辆)
   - ASP提升空间大(从4,800元→8,000+元)
   - 2025年汽车电子收入预计超300亿元(增长50%+)
   - 毛利率高达12-15%

2. **AI创新周期驱动消费电子复苏** ⭐⭐⭐⭐
   - 苹果Apple Intelligence推动AI终端升级
   - iPad组装份额30%+，收购捷普切入iPhone中框
   - 钛合金等高端零部件渗透率提升

3. **AI服务器新业务打开长期空间** ⭐⭐⭐⭐
   - 与英伟达战略合作
   - 液冷、电源产品通过认证
   - 2025年有望贡献30-50亿元收入

4. **估值吸引力明显** ⭐⭐⭐⭐⭐
   - 当前PE仅16-18x
   - 相对同行折价10-15%
   - 安全边际15-27%

### 三大主要风险

1. 🔴 **大客户依赖**: 对苹果收入占比40-50%
2. 🔴 **毛利率偏低**: 2024年仅6.94%
3. 🔴 **商誉减值风险**: 捷普收购产生43.62亿商誉
