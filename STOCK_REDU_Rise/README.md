## 研究结论

### 退市原因分析

1. **股价低于1美元**
   - 连续30个工作日低于1美元
   - 收到纳斯达克退市警告
   - 股价仅0.479美元 (退市前)

2. **"双减"政策冲击**
   - 学科类培训受限
   - 业务被迫转型
   - 收入大幅下降

3. **持续亏损**
   - 2021年上半年亏损7727万元
   - 转型成本高昂

### 历史财务表现 (退市前)

| 指标 | 2021年上半年 | 2020年同期 | 变化 |
|------|-------------|----------|------|
| 营收 | 5.18亿元 | 2.74亿元 | +89.1% |
| 净亏损 | 7727万元 | 约10亿元 | 亏损收窄 |
| 现金 | 5.108亿元 | - | - |

### 业务转型

**转型方向**:
- 学科英语 → 素质教育
- 瑞思少儿中心
- 瑞思海芽成长空间
- 然点科学馆

## 投资者启示

### "双减"政策影响

瑞思教育是"双减"政策的典型案例：
- 学科类培训业务受重创
- 被迫转型素质教育
- 转型成本高昂
- 最终私有化退市

### 经验教训

1. **政策风险**: 教育行业政策变化影响巨大
2. **业务单一**: 过度依赖学科类培训
3. **转型困难**: 素质教育竞争激烈
4. **现金流**: 烧钱模式下难以持续
