## 核心投资论点

### 看多理由 🟢

1. **AI PC换机周期**: 2025年Windows 10终止支持将驱动企业换机,AI PC渗透率快速提升(2027年预计达60%)
2. **供应链护城河**: "全球资源+本地交付"模式,30个制造基地,Gartner全球供应链第10位
3. **ISG业务爆发**: AI服务器业务同比增长63%,连续季度盈利,成为第二增长曲线
4. **SSG高利润**: 运营利润率超21%,解决方案业务占比60%,提供稳定现金流
5. **估值合理**: PE约10倍,低于历史均值,PC业务见底复苏

### 看空理由 🔴

1. **PC市场萎缩**: PC出货量从2021年3.43亿台降至2023年2.47亿台,长期增长乏力
2. **净利率偏低**: 销售净利率仅1.6%,盈利能力较弱,缺乏定价权
3. **ISG历史亏损**: ISG业务长期亏损,盈利持续性待验证
4. **竞争激烈**: PC行业竞争激烈,产品同质化严重,难以提价
5. **宏观风险**: 全球经济放缓、地缘政治、供应链扰动等外部风险
