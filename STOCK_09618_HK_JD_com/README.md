## 🎯 投资亮点

### 优势
1. **物流网络壁垒**: 1600+仓库，66万+员工，"211限时达"行业标准
2. **用户规模**: 7亿+年度活跃用户，用户购物频次+30%
3. **供应链效率**: 客户库存周转天数33.3天，对标Costco
4. **现金流稳定**: 经营现金流700亿，应付账款占资能力强
5. **股东回报**: 2024年回购36亿美元 + 分红14.4亿美元

### 风险
1. **盈利能力弱**: 净利率2.33%，远低于同行(拼多多24.3%、阿里12.6%)
2. **新业务烧钱**: 外卖业务2025年亏损148亿，短期难以盈利
3. **竞争加剧**: 拼多多低价冲击、抖音电商崛起、社区团购分流
4. **增长放缓**: Q4收入增速1.5%，国补退坡后压力加大
5. **重资产模式**: 资产周转率低，ROE仅8-10%
