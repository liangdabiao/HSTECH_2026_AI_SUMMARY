## 投资要点速览

### ✅ 核心优势 (Thesis)

1. **华为核心供应商** - 连续多年获得华为核心供应商奖，5.5G基站主板核心供应商
2. **技术实力突出** - 高频高速PCB、HDI板技术领先，通信设备PCB市场占有率约30%
3. **行业景气上行** - 5G建设持续推进、AI服务器需求爆发、汽车电子快速渗透
4. **产能扩张积极** - 多地产能布局，支撑未来增长

### ⚠️ 核心风险 (Risks)

1. **客户集中度高** - 前五大客户营收占比69-74%，华为作为第一大客户存在依赖风险
2. **估值不便宜** - PE (TTM) 54倍，处于历史较高水平
3. **毛利率承压** - 原材料价格波动、行业竞争加剧
4. **技术迭代风险** - PCB技术快速演进，需持续高研发投入
